# 日内交易五步流程 — Umar Ashraf

## TL;DR
Umar的超短线日内交易固定五步流程：交易前准备（设定切实目标+例行程序）、宣誓承诺、清晰交易计划（关键区域价格行为）、设定不在乎的仓位与风险、记录与复盘。强调心理因素占90%。每笔交易只吃确定性高的一段走势。

## Setup / 形态
- 资产：美股（亚马逊等大盘股）
- 时间框架：超短线日内、1分钟图表
- 风格：高杠杆超短线价格行为交易员

## Entry Rules / 入场
**第一步：交易前准备** (00:45)
- 设定具体且现实的目标，例如：
  - 通过优化策略提高胜率
  - 改善每笔交易风险回报比
  - 像退学炒股一样靠交易利润维持生活开销
- 避免不切实际想法（1000美金本金年底翻到1000万；止损1U止盈1000U；买山寨币却幻想买到第二个比特币）
- **每天交易前例行程序**：晨跑、看经济数据/新闻消息 — 取决于个人习惯
- 有规律生活减少市场波动对情绪的影响

**第二步：发誓言** (02:14)
- 类似销售人员每天给自己打鸡血
- 加强对目标和计划执行承诺
- 把毒誓写下来放在随时能看到的地方
- 时间久了对止盈止损越来越坚定
- 前两步都与情绪相关，**心理因素占长期盈利的90%**

**第三步：清晰交易计划并坚决执行** (02:53)
- 不要预测行情，根据价格行为做出反应
- Umar是价格行为交易员，**经常在关键区域进行突破或波段操作**
- 在支撑、阻力位和供给需求区域附近交易
- 提前分析大级别周期的关键位置并标注图表
- 价格接近关键区域时准备交易
- 交易前计划好：
  - 进场时机
  - 止损设置（不容易被打到+走势不如预期时限制损失）
  - 止盈位置（大概率成交+防止提早获利了结）
  - 价格反转后如何操作

**亚马逊空单实例** (03:48)
- 1分钟图表观察价格从127反弹到130附近的大周期供应区位置
- 没有放量突破迹象，价格上涨疲软
- 在供应区附近设置空单
- 止损：突破130.5
- 止盈：下方的顶底转换结构位 = 128.8附近
- 走势如预期下跌到128.8附近止盈出场
- **Umar从来只吃确定性高的一段走势**，这是他能稳定盈利的关键

## Exit & Stop Loss / 出场止损
**第四步：仓位规模与风险范围** (04:36)
- 选择不会在意的仓位规模与风险范围 → 交易状态最好
- 拜恩斯案例：每笔交易风险**固定1000美元**，从不提高或降低，即使净资产达数百万
- **冒1000美元风险去赚2000美元，亏损不会触动情绪**

## Risk Management / 仓位
- 固定每笔交易的风险金额
- 不因为账户增长而提高风险敞口
- 拒绝把1000提到更高风险的诱惑

## Examples / 实例
- 亚马逊空单：127反弹到130供应区，无放量突破 → 入场空单 → 130.5止损 → 128.8止盈

## Caveats / 注意
- 不要试图通过提高仓位或扩大风险回报比来加速积累
- 贪婪和恐惧是超短线高杠杆选手必须克服的
- "1500万美元绝大部分来自近几年，前期盈利可能只占10-20%"
- 有人十多年交易经验可能都不及别人两年的水平
- 各种经济金融交易知识很枯燥，记录每笔交易很辛苦 — 但必须坚持

**第五步：记录和复盘交易** (05:21)
- 持续优化改进交易方式
- 经常复盘图表和分析以往交易
