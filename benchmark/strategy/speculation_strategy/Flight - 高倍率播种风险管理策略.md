# 高倍率播种风险管理策略 — Flight

## TL;DR
韩国传奇交易员 Flight 的核心仓位管理框架。每次只用账户**1%**资金 × **20倍杠杆**（= 持有2%价值的标的），单日不超过 10 笔、3 个标的，单标 2% 止损（即损失账户的 0.04%）。把每次交易当"播种"，结合高确定性策略博取超额收益。

## Setup / 形态
- **核心理念**：用高杠杆 + 严格控制每次投入资金比例 + 高确定性策略 → 极端行情不大亏，同时博大利润空间
- **故事背景**（00:00:38-02:21）：
  - 2018年200万韩元爆仓
  - 借50万韩元（约$370）二番战
  - 1个月赚到2500万韩元
  - 5个月做到2亿韩元
  - 2019/1月垃圾行情损失一半仓位，最后剩2000万韩元
  - 认清全仓进出风险，设计了"小资金大杠杆 + 严格分仓"的新方法
  - 提现1400万存银行，仓位只留600万开始飞行挑战

## Entry Rules / 入场
**仓位框架（以10000美元账户、比特币为例）**（00:02:51-03:50）：

| 参数 | 数值 |
|---|---|
| **账户本金** | $10,000 |
| **单次投入仓位比例** | **1%**（$100） |
| **杠杆** | **20倍** |
| **单次持有价值** | $2,000 BTC |
| **单次开仓标的数量上限** | **3 个** |
| **每天开仓次数上限** | **10 次** |
| **单标的止损** | **2%**（损失 $40 / 0.4% 账户） |
| **单日总损失上限** | **$400**（占账户4%） |

**入场触发**：
- 符合任一高确定性策略入场信号（见 Flight 其他策略文件）
- 单次开仓最多 3 个标的，控制集中度

**逐仓模式**：
- 也可选"爆仓即止损"（开仓模式设置成**逐仓**）
- 同样保证一天的仓位损失不超过 $400

## Exit & Stop Loss / 出场止损
- **止损**：单标 2%（或选爆仓即止损的逐仓模式）
- **止盈**：
  - 分批止盈
  - 或到达预期价格直接获利了结
- 高确定性策略时经常可以捕捉到大利润空间

## Risk Management / 仓位
- **小资金对情绪触动小**（00:03:50）
- **必须经常提现**（00:03:56-04:11）：
  - 不要把账户资金堆积过大
  - 将赚到的钱存到银行
  - 尤其是出现巨大盈利或累计可观数额之后
  - "否则一切都是泡影"
  - "切记这个行为非常的重要"

## Examples / 实例
- Flight 实战：$370 → $750万（账户翻 20,270 倍）
- 6 个月入门期 + 多年实践
- 提现 1400 万韩元存银行，仅留 600 万作为交易资金

## Caveats / 注意
- 必须配合**高确定性策略**使用，否则20倍杠杆会快速归零
- 单日 10 次开仓+3 标的+2%止损 = 累计单日损失上限 $400
- **经常提现是策略的不可分割部分** — 不提现 = 一切都是泡影
- 高杠杆爆仓即止损要求资金管理极严
- 该策略让 Flight 从 200 万爆仓中重生
