# 价格行为三步框架 — 频道编辑

## TL;DR
通过三步法把价格行为系统化：第一步用N字结构判断大周期趋势；第二步标记价值区域（密集成交区/供需区/波段高低点/斐波那契回撤/整数关口）；第三步切换到小级别图表确认反转形态或MACD背离再入场。顺大势逆小势，回调到价值区域时入场。

## Setup / 形态
- 时间框架组合：15分钟（小）+1小时（中）+4小时（大），适合短线 (01:01)
- 多个市场可用：比特币、欧元/美元等
- 核心思想：交易由买入卖出组成订单，订单构成K线，K线组合成N字结构，N字结构叠加形成趋势 (01:11)

## Entry Rules / 入场
**第一步：判断大周期趋势** (01:09)
- 上升趋势 = 多个"更高的高点 + 更高的低点"的N字结构组合
- 下跌趋势 = 多个"更低的高点 + 更低的低点"的N字结构组合
- 例如4小时图表如果是多个更高高点+更高低点的N字 → 上升趋势 (01:47)
- 原则：顺大势逆小势 — 大趋势上升时，选择价格回调时入场做多 (01:56)

**第二步：标记价值区域** (02:12)
- 价值区域类型：
  - 密集成交区
  - 供给需求区
  - 明显的波段高低点
  - 斐波那契的重要回撤位置
  - 整数关口（心理支撑位）
- 这些位置因为依据它们做单的资金多，附近会分布大量未成交订单，是潜在的价格停止区域，盈亏比最好的交易点位 (02:30)

**第三步：小级别验证** (02:51)
- 切换到1小时或15分钟图表观察是否出现价格反应
- 反转形态：头肩底、双底、长下影线PINBAR
- 指标信号：MACD背离（原本强势的一方在该区域变弱）
- 多单实例：价格回调到价值区域时，1小时MACD出现底背离 → 入场 → 15分钟出现头肩底 → 突破颈线处加仓 (03:32)

## Exit & Stop Loss / 出场止损
- 止损：设在价值区域的下方（多单）或上沿（空单）(03:47)
- 第一止盈位：上一个波段高点附近 (03:51)
- 当4小时级别也走出头肩底形态时，剩余部分仓位可大胆持有，博取更高利润 (03:55)
- 价格继续推进时把止损上移到入场价（避免盈利转亏损）(04:02)
- 空单止盈可选分批止盈：震荡区间的底部出一部分，前低位置出全部仓位 (04:55)

## Risk Management / 仓位
- 没有明确量化规则，仅强调"加仓一部分"在小级别确认形态时

## Examples / 实例
- 比特币/美元多单实例：4小时上升趋势 → 1小时MACD底背离 → 15分钟头肩底突破颈线加仓 → 4小时头肩底出现继续持有
- 欧元/美元空单实例：4小时N字结构呈下跌趋势 → 1小时双顶+MACD顶背离 → MACD实心转虚心柱入场空单 → 分批止盈在震荡区间底部、前低位置

## Caveats / 注意
- 价值区域的有效性必须由小级别价格反应/反转形态/指标信号验证才入场
- 不要在没有小级别确认信号时盲目入场价值区域
- 单纯依据K线而无视其他指标过于片面
