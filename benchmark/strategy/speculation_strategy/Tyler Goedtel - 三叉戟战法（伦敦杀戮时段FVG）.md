# 三叉戟战法（伦敦杀戮时段 + FVG + 顺势侵入） — Tyler Goedtel

## TL;DR
从美国电梯井走出来的Tyler用 **胜率>90%** 的"三叉戟战法"3个月赚40多万美金。盈亏比极高——常规1:10-1:20，高达1:40-1:50。核心是在 **伦敦杀戮时段** (NY凌晨3-6:30) 找4条EMA(5/9/13/21)整齐堆叠的强多头/空头结构 + 时区内出现的 **FVG** + 三叉戟形态 (十字星影线穿过FVG中点 + 下一根K线收盘在十字星高点下方) 同时满足。信号极少——一个品种一年可能就几次机会。

## Setup / 形态

### 核心理念
- "用最短的时间，在动能最集中的时段，出手捕捉最容易产生大收益的机会" (01:05)。
- 伦敦杀戮时段对 Tyler 来说是黄金交易时间 (01:13)。

### 交易时段 (01:18)
- **纽约时间 凌晨 3:00-6:30** (TradingView锚定纽约时间，需自行换算成本地)
- 根据Tyler多年回测，这3个多小时几乎就是市场送钱的时段。
- 各位可根据品种调整时间：比如BTC经常在北京时间晚10点-1点或上午7-10点出现行情。
- 指标 (01:43)：搜 **Trident Pattern [SpeculationLab]** 或点视频主页直接添加到图表，可在指标设置中自由设时段。

### 标的选择 (02:09)
- Tyler 总是做多，专注 **自然看涨的资产**：黄金 / 比特币 / 纳斯达克。
- 在日线确认趋势 → 30分钟图找入场 (02:15)。

### 指标设置 — 5条EMA (02:21)
- **EMA 5 / 9 / 13 / 21 / 200**
- **EMA200 作用**：在日线图表中指示趋势——价格在EMA200上方 = 多头趋势 (02:30)。
- **30分钟图**：EMA 5/9/13/21 这4条必须 **均线堆叠，依次从上到下排列、无任何交叉缠绕** (03:01) = 强劲看涨结构。

### 趋势确认 — 辅助信号 (02:36)
- 指标搜索 **Bull Trading one minute easy scalping**
- 添加到图表后打开设置，**时间框架 = 日线**
- 通过K线颜色区分强度和动量：**强劲看涨 = 绿色**
- 是 EMA200 之外的额外谨慎信号。

### FVG (公允价值区/Fair Value Gap) 要求 (03:11)
- FVG 必须出现在 **EMA5 均线堆叠的上方**——这时候的FVG才具有交易价值 (03:15)。
- **FVG 形成时间** 必须在 **凌晨 2:30 - 4:00** 之间 (03:18)，即交易时间的前半段，可放宽半小时。
- 这是为了说明动量缺口的产生来自动量最集中的时段。

### 三叉戟形态 (Trident Pattern) (03:33)
- **第一根**：十字星 (doji)，影线要 **穿过或触及 FVG 的 50% 位置** (03:37)。
- **第二根**：必须 **收盘在十字星的高点下方** (03:43)。
  - 如果收盘在高点上方 → 形态无效。
- 这个价格行为有专门名字叫 **"顺势侵入"** (Inversion Fair Value Gap 性质的入场触发) (03:52)：用来判断FVG是否被合理回补，强劲多头资金使价格快速反弹，显示市场资金希望价格进一步走高。

## Entry Rules / 入场

### 多头入场（4条件全部满足）(04:03)
1. **日线趋势多头** (价格在EMA200上方；可选 Bull Trading 指标K线为绿色)。
2. **30分钟图 EMA 5/9/13/21 整齐堆叠**，无任何交叉缠绕。
3. **时区内** (NY 凌晨3-6:30) 的 **FVG** 出现在EMA5堆叠上方 且 FVG 形成时间在 NY 凌晨2:30-4:00。
4. **三叉戟形态成立**：十字星影线穿过/触及 FVG 50% 位置 + 下一根 K 线收盘在十字星高点下方。
5. 30分钟图在三叉戟收盘后出现入场信号 → **在第二根 K 线 (顺势侵入K线) 收盘价顺势进场** (04:12)。

## Exit & Stop Loss / 出场止损

### 止损 (04:16)
- 可以放在 **形成 FVG 那根 K 线最低点的下方**。
- 也可以是 **波段低点下方**。
- 对于美元相关货币对，Tyler 常用 **约10个点** 的止损 (04:23)。
- **黄金不用硬止损** (04:27)：等K线收盘低于某一个特定的K线后再做决定——因为黄金波动性很大，硬止损容易被扫流动性的影线清理掉。

### 止盈 (04:41)
- 设定在日线图，**尽可能长时间地顺着趋势持有**。
- 不是百分百机械化，**有一定自由裁量性** (04:45)：
  - 当 EMA 交叉，或出现大的看跌K线时才平仓。
- 退出靠时间和经验积累的直觉 + 以往交易数据回馈建立的自信 (05:00)。

## Examples / 实例

### BTC 案例 (05:04)
- 价格运行在 EMA200 上方 → 趋势看多。
- 切换到 30 分钟图，均线堆叠整齐 → 动能很强。
- 在 NY 凌晨3点至NY凌晨3:30(原文"纽约十点到一点半"似有口误) 一个干净的三根K线缺口出现。
- 随后形成的十字星影线插入中点。
- 后一根 K 线收盘也低于十字星高点。
- 这根K线就是关键K线，三叉戟形态成立 → 关键K线收盘价顺势进场 → 盈利幅度非常可观 (05:35)。

## Caveats / 注意
- **信号极少**：一个品种每年可能就几次机会 (05:48)。
- "有的品种胜率和盈亏比可能没有他说的那么高" (05:43) ——需要多发掘合适的天然看涨品种。
- 不同品种的高波动时段也不同，要自己观察总结 (05:53)。
- 主播为策略编的对联：**"右联：关键时段顺势找缺口；左联：用三叉戟抓住大行情；横批：信号少，且得等"** (05:58)。
- TradingView 指标固定锚定纽约时间，北京时间需自己换算。
- 必须严格的日线EMA200/30分钟均线堆叠/时区FVG/三叉戟四共振，缺一不可。
