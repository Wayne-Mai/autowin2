# 关键位+催化剂+归零仓位法策略 — Brando

## TL;DR
在美生活的越南裔交易员，6年用 $6,000 做到 $1,000万+（自称）。以波段交易为主，在日线/周线大级别的关键支撑阻力位置 + 同期出现的重大新闻或数据催化剂 + 入场确认信号 三重共振时入场期权，使用"归零仓位法"管理风险。

## Setup / 形态
- **核心三要素**：**关键水平 + 催化剂 + 确认信号**
- **交易工具**：**周合约 / 月合约的期权**（00:02:03-02:07）
- **风格**：以波段交易为主
- **关键位查找级别**：**日线和周线图**（大级别）

## Entry Rules / 入场

**步骤1 — 找关键水平**（00:02:07-02:19）：
- 过去的重要高低点
- 重要支撑与阻力位
- 整百整千的心理整数位（如S&P 5000、5700）
- 这些位置都有很强的价格吸引力

**步骤2 — 等催化剂**（00:02:19-02:41）：
- 真正有效的入场机会必须**伴随强有力的催化剂**
- 常见催化剂：
  - 美联储会议（FOMC）
  - CPI 数据
  - 非农数据
  - 重点公司财报（英伟达、苹果等）
  - 关税
  - 选举等重大地缘政治事件
- **只有当技术面和催化剂同时出现的时候才考虑入场**

**步骤3 — 等确认信号**（00:02:41-02:55）：
- 关键位置出现强烈反弹或拒绝
- 关键价格附近持续震荡或重新站稳
- 催化剂出现后市场情绪明显转向
- "这一步很关键，是在等市场验证你的判断"

**周计划流程**（00:03:56-04:25）：
- 每周开始前先搞清楚本周有哪些重要新闻 / 数据发布
- 列出周一到周五的具体安排
- 重点围绕大事件计划交易
- 美联储会议前：只做短线（大行情大概率在会议结束后才出现）

**理解波动性**（00:04:25-04:43）：
- 通常美联储会议之后标普500大约出现 **1.7% 的波动**
- 市场整体波动性大约增加到超过 **20%**
- 知道这些对**期权交易**非常重要

## Exit & Stop Loss / 出场止损
**归零仓位法（核心）**（00:04:51-05:30）：
- **把仓位大小控制在即使完全亏损也能接受的范围内**
- 不是把止损放的特别紧凑
- **对比**：
  - 错误做法：冒 $5,000 风险设置 20% 止损
  - 正确做法：将总仓位缩小到 **$1,000**，这 $1,000 作为最大风险
- "亏损归零自己也能够接受"
- **爆仓即止损**
- 用更大的风险承受力瞄准更高倍的收益

**优点**：
- 培养情绪控制
- 避免被过早洗出局
- 期权波动剧烈（可能前一天跌60%、第二天暴涨300-400%），紧止损反而被洗

## Risk Management / 仓位
- 归零仓位法：单笔仓位 = 单笔最大可接受亏损
- 严格分仓，不重仓单一标的
- 一个月理想机会不多 → 需要极大耐心

## Examples / 实例

**实例1 — 2024年5月 标普500**（00:01:11-02:02）：
- 标普500 多周盘整后回调到 **5,000** 整数价位（周线关键支撑）
- 价格首次测试 5,000 后，连续形成两个更高周线低点 → 潜在反转形态
- 同时英伟达公布超预期财报，股价暴涨 25%+
- 通胀数据降温迹象 + 劳动力市场稳定
- Brando 在 **$5,150-5,160 区间买入看涨期权**
- 之后标普500 两周内涨 200+ 点
- 单笔盈利 **超过 $100万**

**实例2 — 2024年11月美国大选**（00:03:03-03:46）：
- 标普500 选举前几个月在 **5,700** 下方盘整
- 5,700 阻力位连续多周成为天花板
- 大选后特朗普连任带来增长预期+放松管制
- 价格直接跳空企稳在 5,700 **上方**（技术突破确认）
- Brando 在站稳 5,700 后买入看涨期权
- 之后标普500 一个月内涨 **400+ 点**，最高接近 6,147

## Caveats / 注意
- 频道主对 Brando 收益数字有疑虑（"我搜不到差不多的交易记录"）但策略本身被欧美圈认可（00:00:17-00:42）
- 适合**有耐心等待高确定性机会**的交易员
- 一个月理想机会可能不多
- 在真正关键位置 + 催化剂吻合时，往往**市场分歧很大，执行压力可能超过承受能力**
- 必须在心理和技术上都做好准备
- 新手经常在没有催化剂时交易 — 经验丰富的交易员总是问"什么才是真正推动价格运动的催化剂？这个催化剂够不够分量？"
