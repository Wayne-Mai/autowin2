# 屯币宝一篮子轮动策略 — Speculation Lab（介绍 OKX 工具）

## TL;DR
屯币宝是 OKX 的一种"一揽子币种 + 按比例自动再平衡"的量化工具。通过设定多个币种的目标仓位，当某个币种因上涨偏离目标比例时自动卖出涨多的、买入涨少的，实现自动高抛低吸。

## What It Measures / 它衡量什么
- 一揽子资产中各币种仓位比例相对目标比例的偏离度
- 偏离触发再平衡 → 卖出相对涨幅大的，买入相对落后的
- 本质：分散风险 + 自动高抛低吸 + 减少对单一币种涨跌的主观预测

## Parameters & Settings / 参数设置

### 双币轮动示例（01:25-01:57）
- 初始：比特币 50% + 以太坊 50%，各投入 500U
- 比特币涨到 800，以太坊涨到 600
- 策略自动卖出 100U 比特币买入以太坊
- 保持两种币仓位比例相等，资产从 1000U → 1400U

### 在 OKX 创建步骤（02:01-04:43）
1. **入口**：OKX 策略广场 → 组合套利菜单 → 屯币宝
2. **币种配置**：
   - 智能组合（OKX 回测 24 小时收益优选）
   - 玩家组合（长期运行表现）
   - 自定义参数
3. **币种数量**：最多可选 10 个币种组合
4. **币种选择建议**：
   - 同板块、大市值的币种
   - 不要选山寨小币
   - 推荐：比特币、莱特币、以太坊
5. **比例分配**：智能分配 / 平均分配 / 自定义
   - 视频示例：BTC 50% + LTC 30% + ETH 20%
   - 比特币是行情龙头 → 50%
   - 莱特币减半行情 → 30%
   - 以太坊补涨潜力 → 20%

### 两种模式
**比例平衡**（03:17-04:20）：
- 按币种价值变化比例触发平衡
- 策略机器人每 5 分钟检查一次持仓比例
- 设定偏离阈值（如 ≥ 10%）触发再平衡
- 示例：10,000U 投入（BTC 5000 / LTC 3000 / ETH 2000，比例 50/30/20）
- BTC 涨至 7500U → 比例变 60/24/16，BTC 偏离 10% 以上
- 自动卖出 1250U BTC → 买入 750U LTC + 500U ETH
- 持仓回到 6250/3750/2500（即 50/30/20）

**定时平衡**（04:20-04:34）：
- 按固定时间周期（分钟/小时单位自由设置）平衡
- 偏离程度超过默认 3% 触发

### 投入金额
- 视频示例：1000U
- 创建后自动运行

## How To Read / 怎么解读
- 大行情来临时难押注最强币种 → 用一篮子分散
- 资金溢出板块轮动时，自动高抛低吸优化收益
- 与网格策略类似，但屯币宝可买入更多币种（网格只能两个币种间互换）
- 如果配置 USDT + ETH + BTC → 屯币宝变成币币网格策略（06:16-06:23）

## Trade Setups Using It / 配套的入场出场
- 长期被动策略，不需主动入场
- 资金从交易账户隔离 → 独立运行
- 触发条件由策略机器人自动执行

## Caveats / 注意

**三大要点**（04:54-06:07）：

1. **币种配置要清晰**（04:54-05:27）：
   - 必须选长期有价值的大市值币种（BTC、主流币）
   - 不要选山寨垃圾币
   - 否则会出现"劣币驱逐良币"：机器人不断卖出真正优质币去填补下跌的假优质币

2. **同板块/相近币种**（05:27-05:54）：
   - AI 板块轮番拉涨时只选 AI 板块
   - 或选 BTC + 市值前几的主流币
   - 不要"不着边际瞎选" → 否则会被卖飞的强势币继续涨，留下乌龟币

3. **资金隔离风险**（05:54-06:07）：
   - 投入资金会从交易账户隔离出去
   - 注意原有交易账户仓位是否因此承担风险
