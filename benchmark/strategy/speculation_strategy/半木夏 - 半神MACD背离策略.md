# 半神MACD背离策略 — 半木夏

## TL;DR
半木夏用MACD柱状图连续波峰背离 + ATR止损的高胜率策略，1万本金做到5000万。规则极其清晰：MACD零线下方/上方找2-3个波峰相差较大的连续MACD柱（最好3个），对应K线低点连续降低（多）或高点连续抬高（空），MACD实心柱转虚心柱的第一根对应K线即入场K线，ATR做止损。

## Setup / 形态
- 资产：加密货币（比特币/美元）
- 时间框架：15分钟级别为例
- 工具：仅需2个免费TradingView指标 — MACD柱状图、ATR

## Entry Rules / 入场
**指标设置** (00:56)
- MACD：快线长度 = 13，慢线长度 = 34（金叉死叉本系统用不到，可取消）
- ATR：参数 = 13

**多单规则** (01:43)
1. 在MACD**零线下方**，找到2个或3个波峰相差较大且连续升高的MACD柱状图（**3个最好**）
2. 关键细节："**一定要找波峰相差较大的连续背离**" (01:12) — 99%的人都不知道这个细节
3. 找到与MACD对应的K线，看K线低点是否连续降低
4. 条件满足：**MACD实心柱转成虚心的第一根柱子所对应的K线 = 入场K线**
5. 入场K线收盘价开单

**空单规则** (02:23)
1. 在MACD**零线上方**找到3个波峰相差较大且**连续降低**的MACD柱状图
2. 对应K线高点是否连续抬高
3. MACD实心柱转虚心柱所对应的K线 = 入场K线
4. 也可以是零线上方两个相差悬殊+波峰连续降低的MACD柱状图

## Exit & Stop Loss / 出场止损
**多单止损** (02:09)
- 止损位 = 入场K线低点 − 对应的ATR数值
- 例：ATR数值为173

**空单止损** (02:40)
- 止损位 = 入场K线高点 + ATR数值

**止盈** (02:45)
- 风险回报比1:1、1:2，甚至更高的位置**分批获利**
- 默认目标1:2
- 实际案例中实现了1:8的盈亏比

## Risk Management / 仓位
- 视频未详述仓位管理规则，仅强调风险回报比要够

## Examples / 实例
- 比特币/美元15分钟交易对，三波峰多单：MACD三个波峰相差较大且连续升高 → K线低点连续降低 → 实心转虚心入场 → ATR=173做止损 → 风险回报1:2快速完成
- 两波峰多单实例：零线下方两个波峰相差悬殊+连续降低（注：转录此处可能口误"连续降低"应作"连续升高"看待）
- 空单实例：吃到了1:8盈亏比

## Caveats / 注意
- 关键细节是波峰必须**相差较大**，不能是普通背离
- 三个波峰比两个波峰胜率更高
- 一定要进行广泛的盘面测试，找到最适合这套系统的交易品种再去参与
- 不同品种回测结果不同
- 该策略是半木夏早期版本，2年后他演化到了周期+流动性+价格行为的终极系统
