# 经典突破模型 — Marios Stamatoudis

## TL;DR
Marios（USIC 2023第三名，291%收益）三种模型之一。前置条件：必须有基本面燃料支撑。规则：价格在日线MA10/20/50/100/150均线上方，前期强劲上涨后长时间横盘震荡，最终放量突破即入场。出场用MA10或MA20追踪。

## Setup / 形态
- 资产：股票（也适用比特币、生物科技、加密市场等）
- 时间框架：日线
- 工具：日线5MA（MA10、MA20、MA50、MA100、MA150） — TradingView指标搜索栏搜"5MA"

## Entry Rules / 入场
**前置条件：除草留花（基本面筛选）** (00:44)
- 每天筛选有基本面支持的强势品种：
  - 财务信息异常好的
  - 有重大事件发生的
  - 业务结构有重大变动的
  - 加密市场有大流动性进入的
  - 生物技术有新药或批准的
- 每天查看头条新闻和热门行业产业新闻
- 没有新闻+没有基本面支持 → 上涨骗炮概率大，不持久

**入场规则** (01:30)
1. **价格在日线MA10/20/50/100/150均线上方**
2. 查看标的过去几周内上涨情况，确定市场情绪和状况
3. **最好条件**：标的前期价格是强劲的上涨趋势 → 开始长时间横盘震荡
   - 震荡时间越长流动性消耗越多
   - 震荡幅度越来越小
4. **最终的放量突破** = 市场参与者对未来价格上涨的共识反映 → 入场

## Exit & Stop Loss / 出场止损
**出场规则** (01:55)
- 用**MA10或MA20均线追踪出场**
- 当标的当天**收盘价低于均线时部分平仓**
- 剩余仓位可以：
  - 上移到**盈亏平衡位置**设置出场
  - 或在**MA100均线**追踪出场

## Risk Management / 仓位
- 同时持有6-7个品种（任何时候不超过15个）
- 平均仓位规模 = 账户资金的13-16%
- 每笔订单风险 = 账户资金的0.25-0.4%
- 计算：30%胜率下，约70%概率会在50笔样本中连续输掉10笔
- 若每笔冒1%以上风险，账户可能缩水10% → 资金回撤会影响交易舒适度
- **止损一定要小，风险回报比一定要大，宁可不做也不能冒太大风险**

## Examples / 实例
- 2023年全年交易约500次（平均每天2次），全部胜率32%，收益率291%，风险回报比平均超过1:5
- 大部分利润不是平均获得，**只来自10-15笔交易**（不对称回报很正常）

## Caveats / 注意
- 过度交易让人精疲力尽
- 没有燃料推动的上涨很难持久
- 不对称回报是正常现象
- 持仓越多情绪上越容易犯错
