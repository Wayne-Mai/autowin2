# 阶段前高前低反弹策略 — 频道编辑

## TL;DR
"既简单又实用"的策略：行情中的价格跟超市里的商品一样——前低附近往往堆积大量未成交买单(错过低价的买家会再次抢购)，前高附近往往堆积卖单(套牢者怕再错过出货机会而挂卖)，订单堆积过多就容易使价格反转或反弹。技术分析的目的就是找到价格停止的位置，而能让价格停止的，莫过于订单累积过多的地方。

## Setup / 形态

### 核心类比 (03:59)
- 行情价格 = 超市商品。
- 超市牛肉打一折时购买人就比不打折多，甚至有人因价格便宜而大量囤货——导致后来想买的人没货。
- 牛肉再次打折，前面没抢到的人再见到就大概率开始抢着买。

### 交易市场逻辑 (04:17)
- 价格跌到之前一段走势的低点附近 → 经常会有错过低价位的人选择买入。
- 尤其是第一次到达这个区域时，关键位置附近的买单和平空订单会明显多。
- 订单堆积过多 → 容易使价格出现反转或反弹。

### "前高" (供给区) 与 "前低" (需求区) (04:33)
- 前高的概念类似前低：手里有筹码的人不认为价格会再创高点，担心上次没卖、这次再错过就不好卖了，所以选择挂单卖出。
- 这种位置附近的抛盘分布较多，会形成较大阻力 (04:42)。

### 技术分析的核心目的 (04:45)
- 要找到价格停止的位置；能让价格停止的莫过于订单累积过多的地方。
- 当价格再接近这些位置时，这些共识和未成交订单就会对价格产生影响 (04:53)。

## Entry Rules / 入场

### 通用规则
- 价格再次接近前低 → 做多（预期反弹）。
- 价格再次接近前高 → 做空（预期受阻）。
- **特别强调**：第一次回到关键位置时反应最强 (04:24)。

### 入场点
- 视频未给精确入场触发，可在前高/前低区域出现K线反转形态(吞没、长影、双顶/双底)时入场。

## Exit & Stop Loss / 出场止损
- **止损**：前高/前低区域之外。
- **止盈**：可参考下一个前高/前低或固定比例。

## Risk Management / 仓位
- 第一次到达的位置反应最强 → 集中仓位；多次回访逐次减仓。

## Caveats / 注意
- 第一次接近时反应最强；同一个前高/前低被反复测试效果减弱。
- 与VWAP / 供给需求区 / VPVR密集成交区结合使用效果更佳。
- 此策略可作"过滤层"——其他入场信号必须落在前高/前低附近才更可靠。
