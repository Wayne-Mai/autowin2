# 主导主题一等股长阳突破短线策略 — Hong Inki

## TL;DR
韩国短线交易员Hong Inki把15万韩元做到10亿的暴利策略。核心是先找出当天主导市场主题的"一等股"（涨幅最大），再等它放量突破历史新高（最好6个月内或全历史）并收长阳线，然后在长阳后的两天内集中交易。第三天动能减弱就果断离场。

## Setup / 形态
- 资产：韩股/币圈均适用
- 时间框架：日线 + 盘中（11点和下午1点等时段）
- 工具：交易额/成交量排行榜

## Entry Rules / 入场
**第一步：识别市场主导主题** (01:33)
- 观察交易额排名前15位、涨幅超过5%的标的
- 如果前15位有多只涨幅超过5%且属于同一主题 → 这就是当天的主导主题
- 前15只没确认则可扩大范围，越符合条件效果越好；如果没有，**一定不能勉强**
- 例：2023年8月31日成交量排行榜中Posco DX、彩虹机器人、Neuromeca都是机器人主题 → 机器人是当天主导主题

**第二步：只交易主导主题的一等股** (02:02)
- 只交易涨幅最大的一等标的（不交易涨幅小的二等）
- 涨幅大 = 动量足、关注度高，下跌也更抗跌（很多人想买入一等标的）
- 例：彩虹机器人是机器人主题的一等股；第二天最高涨12.74%，收盘涨7.46%；同主题的Posco DX跌7.66%、Neuromeca跌1.29%
- 警示：一等股进入涨停时**不要去买二等股** — 一旦一等股解除涨停，资金会从二等流回一等，二等股可能大幅下跌 (03:31)

**第三步：长阳突破新高入场** (03:34)
- "最强大也最好用的突破模型" = 一等标的首次突破历史高点的长阳线
- 关键条件：**巨量 + 涨幅超过10%**
- 新高价定义：突破过去**6个月内的高点**；突破历史最高点更好
- 放量突破时做突破的资金多，共识买盘容易聚集，追涨资金推动价格
- 例：Isupetesis 2023年5月25日突破历史新高涨17.1%，长阳线后两个月上涨236%

## Exit & Stop Loss / 出场止损
**入场时机：长阳线后的两个交易日内集中交易** (04:46)
- **第一天（长阳日次日）**：若主导一等股在长阳线突破前高上方收盘，第二天大多数情况会继续涨；回调时可在支撑位入场，或在突破高点的时候入场
- **第二天 / 长阳后第三天的出场规则** (05:01)：
  - 若出现十字星 / 倒锤子线 / 下跌 → 跌破开盘价1%考虑出场
  - **下跌3% 一定要出场**
  - **上涨5%以上 也一定要离场**（快进快出，高贵的短线交易员）

## Caveats / 注意
- 早期Hong Inki信奉价值投资，把所有积蓄买三星生物制剂，遇财务造假却越跌越买甚至贷款买，最终腰斩爆仓 — 因此转向短线
- 短线确定性更高，可以随时处于最佳状态交易
- 主导主题确认不了时**绝不勉强**
- 第三天即使是最强股也很难继续上涨，热度会逐步减弱
- 时刻谨记自己是高贵的短线交易员
