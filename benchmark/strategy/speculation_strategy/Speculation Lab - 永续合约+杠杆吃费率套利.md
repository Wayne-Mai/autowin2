# 永续合约 + 币币杠杆 吃费率套利 — Speculation Lab

## TL;DR
与"永续 + 现货"套利类似，但用币币杠杆替代现货，等于交易所给你一笔有息贷款放大仓位。前提是费率收益必须大于杠杆利息 + 交易手续费。

## Setup / 形态
- 套利公式：`资金费收益 − 杠杆利息 − 交易手续费 = 套利收益`
- 比第一种"现货 + 永续"多出一个杠杆利息成本
- 适合小资金放大套利效率（如 1000U 开 5 倍杠杆 = 5000U 仓位）

## Entry Rules / 入场
1. 在 OKX 路径：交易 → 策略交易 → 策略广场 → 创建策略 → 组合套利 → 套利下单
2. 选择套利标的，按资金费率从高到低排（03:18）
3. 不同币种存在杠杆限制，要先在右腿（币币杠杆）选择 3 倍杠杆 + USDT 保证金模式
4. 左腿（永续合约）也改成同样的 3 倍杠杆
5. 因为费率为正，空头吃多头给的费率 → 左腿永续合约做空
6. 价格选择市价（限价容易出现左右腿不同时成交的风险）
7. 两腿数量都输入相同（如 1000）
8. 点击双腿下单，策略开始运行

## Exit & Stop Loss / 出场止损
- 退出时机：费率接近 0 或转负前
- 注意：必须在 OKX 永续结算时点（UTC 0/8/16 点）持仓才能收/付费率
- 提前平仓本期不收取/支付任何资金费

## Risk Management / 仓位
- 杠杆放大利润但同时放大利息成本
- 必须确保：`资金费收益 > 杠杆利息 + 开仓平仓手续费`
- 适合稳定大市值币种（BTC、ETH）

## Caveats / 注意
- 小币种短线套利费率高，但 deeper risk（费率方向、流动性差）
- 选 3 倍是因为杠杆限制和成本平衡；倍数越高利息越高
- 必须确认费率历史稳定性，不要追高费率小币种
- 限价单虽然没有左右腿价差，但容易不同步成交，造成"裸单"风险
