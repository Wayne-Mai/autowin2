# Heikin Ashi + Chandelier Exit + ZLSMA 剥头皮策略 — 频道编辑

## TL;DR
源自一个号称96.8%胜率的国外剥头皮策略，使用平均K线图 + Chandelier Exit + ZLSMA 三个指标在5分钟周期捕捉信号。频道主在BTCUSDT回测100次，1:1胜率仅57%，总收益31.9%，并不像宣传那么神奇。

## Setup / 形态
- **图表样式**：双布局上下分屏（00:01:04）。上方Heikin Ashi（平均K线）作为信号指标，下方普通K线为实际交易。
- **指标1 — Heikin Ashi平均K线**：图表样式栏选中"平均K线图 Heikin Ashi"，与普通K线两个收藏快速切换（00:00:35）。
- **指标2 — Chandelier Exit (CE)**：参数改为 1 和 2（00:01:21）。样式栏只留Buy Label和Sell Label，其他用不到的功能勾掉。
- **指标3 — ZLSMA Zero Lag LSMA**：长度=50，颜色白色（00:01:36）。
- **品种/周期**：BTCUSDT 5分钟图表为示例（00:01:46）。

## Entry Rules / 入场
**多头规则**（00:01:46-01:59）：
1. CE指标出现Buy Label买入信号
2. 平均K线的"大部分实体"要处在ZL线（ZLSMA）的上方
3. 两个条件全部满足时，对应的下方普通K线即为关键K线
4. 关键K线的**收盘价**入场多单

**空头规则**（00:02:22-02:43）：
1. CE指标出现Sell Label卖出信号
2. 平均K线的大部分实体要处在ZL线的下方
3. 两个条件全部满足时，对应的下方普通K线即为关键K线
4. 关键K线收盘价入场空单

## Exit & Stop Loss / 出场止损
**多单**（00:02:09-02:21）：
- 止损：近期波段低点附近
- 止盈方式A：平均K线的收盘价跌破CE线时止盈（注意：作者在02:17口误说"CL线"，实指CE线/Chandelier Exit）
- 止盈方式B：风险回报比≥1:1后直接止盈
- 盈利保护：可在1:1附近加保护，价格跌破1:1止盈（00:03:30）

**空单**（00:02:43-02:55）：
- 止损：近期波段高点附近
- 止盈：风险回报比≥1:1后，平均K线的收盘价突破ZL线时止盈
- 或直接在1:1以上止盈

## Risk Management / 仓位
- 不再细化仓位金额
- 频道主提醒：普通账户的手续费支出在这种高频策略下也是很大开销（00:05:00）

## Examples / 实例
- BTCUSDT 5分钟，2023-03-22至2023-04-01连续10个交易日（00:04:23-04:33）
- 100笔回测结果：
  - 1:1止盈胜率：57%
  - 1:2止盈胜率：13%
  - 1:3止盈胜率：5%
  - 1:5以上：暂未发现
  - 100次连续交易总收益：31.9%

## Caveats / 注意
- 频道主给这套策略打分：5分（满分未知，结合语境约为中等）（00:04:52）
- "应用在比特币市场并不那么有利可图"（00:04:56）
- 普通账户手续费会非常侵蚀利润
- 如果要用，建议至少回测500次以上以了解长期表现（00:05:10）
- 平均K线消除了价格波动噪音，但其收盘价与普通K线实际价格有出入，所以实际交易必须以普通K线为准（00:01:04-01:14）
