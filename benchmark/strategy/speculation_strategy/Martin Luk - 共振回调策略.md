# 共振回调策略 — Martin Luk

## TL;DR
22岁香港大学生Martin用胜率仅22%的策略一年内做到969%，夺得2025年美国投资锦标赛冠军。核心思想是日线+周线+小时间周期三个时间框架共振，价格回调到EMA9/EMA21/AVWAP构成的关键区，完成流动性清扫后形成V型反转入场。频繁止损但每次损失很小（1.5%-3%）、盈亏比3-5倍以上。

## Setup / 形态
- 资产：美股（量子计算、AI等热门赛道）
- 时间框架：日线+周线+15分钟/5分钟三周期共振
- 工具：
  - 三条EMA：9、21、50（或用半神MACD指标设为9/21/50的Vegas）
  - AVWAP（锚定成交量加权平均价格）— TradingView侧边栏可找，不占指标额度
  - TradingView

## Entry Rules / 入场
**多单规则**

**第一步：选股和寻找关键支撑** (00:49)
- 在最热流动性最好板块筛选（2025年是量子计算、AI）
- 重点关注：**放量大涨后开始横盘整理**的品种
- 加入观察清单等回调到关键位置
- EMA均线设置 (01:12)：3条EMA分别为9、21、50；不能添加多个指标的用半神MACD设置成9/21/50的Vegas
  - 多头排列 = 9/21/50从上到下排列
  - 9和21线靠得很近时也可以作为支撑位
- AVWAP锚定关键节点起点：公布财报时间点、波段高低点、放量缺口
  - AVWAP紫线 = 该事件至今所有人的平均持仓成本 = 市场对该事件的真实定价
  - 价格长期在AVWAP上方 → 市场资金认可这个事件 → 支撑位

**日线判断** (02:14)
- 股价经过大幅拉升后开始震荡盘整
- 均线呈现多头排列
- AVWAP线区间已收敛得很窄
- 画出EMA9 + EMA21 + AVWAP的区间作为**关键区**

**第二步：切换到周线图确保共振** (02:31)
- 股价在周线的EMA9和EMA21上方
- EMA9必须**倾斜向上**
- 周线和日线趋势指向一致

**第三步：小时间周期找入场点（15分钟/5分钟）** (02:42)
- 看到股价快速突破AVWAP线后**不要着急入场**
- 等价格回到支撑区**完成一次流动性清扫**，又**迅速拉回形成V型反转**
- 判定支撑位有效后：**价格再突破反转K线的最高点** → 市价入场多单
- 反转要快：最好是开盘后不长时间，下跌后出现快速拉升的**暴力反转**
- **温和行情直接放弃**：如果是到美东时间中午才缓慢震荡上行，直接放弃

**空单规则** (03:45)
- 大部分（80%以上）是做多，但也适用做空
- 三条EMA呈现9/21/50从**下往上**排列
- 把阶段高点作为转折点，AVWAP锚定该起点
- EMA9、21和AVWAP距离开始收窄 → 画出关键区
- 周线：股价已跌破EMA9和EMA21，EMA9明显下行走势
- 15分钟：股价开盘后跳空高开 → 向上打进关键区触碰AVWAP → 大阴线反包 → 跌破反转K线最低价时入场做空

## Exit & Stop Loss / 出场止损
**止损规则** (03:15)
- 多单止损：入场K线最低点或当日价格最低点；波动大时设置在开盘价附近
- 空单止损：入场K线高点或当日高点
- **止损幅度控制在1.5%-3%之间**
- 优点：小的试错成本可以有效降低损失
- 缺点：会被经常扫损

**止盈规则** (03:34)
- **盈亏比3到5倍时先卖出三分之一**保留部分利润
- 剩下仓位沿着**日线EMA9移动止损**
- 跌破EMA9就离场
- 做空止盈不同：Martin做空持仓时间都不长，**通常只有两三天**；在加速下跌时主动分批止盈（做空利润空间有限+容易报复性反弹，尤其是强势品种）

## Risk Management / 仓位
- 每次用小试错成本，1.5%-3%止损
- 频繁试仓+止损，需要心理承受能力

## Examples / 实例
- 2025年美国投资锦标赛冠军，969%收益率
- 5单亏4单，胜率22%

## Caveats / 注意
- 这策略很简单但使用起来不轻松，会有频繁试仓→止损→挫败感
- 需要长期坚持并根据自己经常做的品种和周期优化
- 缓慢温和上行的行情直接放弃
- 做空利润空间有限，容易报复性反弹，尤其强势品种
