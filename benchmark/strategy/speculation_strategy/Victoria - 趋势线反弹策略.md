# 趋势线反弹策略（动作线 = 安全线） — Victoria

## TL;DR
趋势线女王Victoria用趋势线策略14笔交易赚了200多万人民币，胜率76.92%，吸引近50万粉丝。反弹策略是认为价格在趋势线上获得支撑/阻力 → 主观预期趋势继续 → **同一条趋势线既作入场的"动作线"也作风险管理的"安全线"**。要求大趋势同向 + 趋势线至少有2-3个清晰触点 + 第一次触点到入场点至少1周。

## Setup / 形态

### 适用标的 (00:42)
- 具有 **强趋势性的大市值标的** 非常适合——价格走势较为规律，经常会在趋势线上反弹；破位后也可以反向操作。

### 工具 (00:58)
- 整个策略只需用到 **动作线** 和 **安全线** 两条线就可以完成入场出场以及移动止损。

### 时间周期 (01:06)
- 4小时周期表现最好——能捕捉最干净的走势，过滤掉低周期频繁的噪音。

### 画法 — 自上而下三周期画动作趋势线 (01:14)

**上升动作线画法 (01:25)**：
1. 周线图：打开周线 → 选 Tradingview **射线工具** (使线在A、B点之后无限延伸)。
2. 找屏幕上看到的 **最低点** = A 点。
3. 找一个 **更高的低点** = B 点（这个低点是价格 **尚未穿过趋势线** 的最新的低点）。
4. 切到日线图：调整和绘制更陡峭的 **动作线** (01:46)；将之前的B作为新A，找下一个最近的低点作为新B。
5. 切到4小时图：重复画线，形成 **趋势线扇形**。

**下降动作线画法 (02:06)**：
1. 周线图：找最高点 = A，更低的高点 = B（同样要求 **尚未穿过趋势线**）。
2. 日线图：将B作新A，找最近低高点作新B。
3. 4小时图：重复。

### 动态调整 (02:38)
- 随行情进展，价格也会穿过最初绘制的趋势线 → 需要定期调整趋势线以适应新的低点/高点 → 消除策略模糊性、保证进出场点位明确清楚。

## Entry Rules / 入场

### 多单入场条件 (3条全部满足) (03:11)
1. **周线/日线大趋势向上**。
2. 趋势线在入场前必须有 **2-3个清晰的触点**。
3. **第一次触点到计划入场点之间至少1周** (03:22)。
4. 满足后，**价格触及趋势线时入场** (03:24)。

### 空头入场条件 (3条全部满足) (04:01)
1. **周线/日线大趋势向下**。
2. 趋势线入场前 **2-3个清晰触点**。
3. **第一次触点到入场点至少1周**。
4. **价格触及趋势线时入场** (04:15)。

## Exit & Stop Loss / 出场止损

### 止损 (反弹策略中动作线 = 安全线) (03:28)
- 趋势线本身即止损线。
- **多头**：价格 **收盘价跌破趋势线** → 果断平仓，说明主体已有向下走的意思。
- **空头**：价格 **收盘价突破趋势线** → 立刻平仓。
- **止损位置要留出足够空间** (03:38)：止损位不能与入场点重合，避免被正常波动扫损。

### 移动止损 (03:50)
- 价格如期向上反弹，中途形成新的有效波段点后 → 调整趋势线做 **移动止损**。
- 空头反过来：价格回落形成新波段点后调整趋势线。

### 止盈 (03:56)
- 可选 **固定止盈** 或在 **关键阻力/支撑区** 附近。

## Risk Management / 仓位

### 单笔风险 (03:42)
- 每次交易的风险尽量控制在账户资金的 **1%-2%** 之间。

## Caveats / 注意
- 必须有 **2-3个清晰触点**——只有1个触点的趋势线不算。
- 必须 **跨周>1周**——避免短时间内多次假触发。
- **大趋势同向** 是基础——逆趋势的反弹策略效果差。
- 趋势线要求 **尚未穿过**——一旦穿过就需重画。
- 高时间框架信号权重高但频率低；低时间框架反之 (05:55)。
