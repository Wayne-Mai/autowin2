# 跳空高开关键阻力做空策略 — Alex Temiz

## TL;DR
前星巴克咖啡师 Alex Temiz 的小盘股做空打法。专做流通量少、市值小、价格低的小盘股跳空高开但缺乏催化剂的情形，在关键阻力区附近做空，止损硬性放在盘前波段高点上方，并按信号评级阶梯式调仓。

## Setup / 形态
- **品种**：流通量少、市值小、价格低的**小盘股**（00:02:05）
- **理由**：这类标的波动幅度大，可以用相对少的资金博取最大盈利空间
- **交易时段**：每天早上6点起床，喝一杯浓缩咖啡后开始交易（00:02:16）
- **每日只交易约1小时**（00:05:26）

## Entry Rules / 入场
**筛选流程**（00:02:23-03:29）：

**步骤1 — 筛选跳空标的**：
- 找到哪些股票有大的涨幅 + 出现了**跳空缺口**
- 找到那些**跳空到关键阻力附近**的股票
- **避开**：跳空到前高附近的股票（继续上行概率太大）

**步骤2 — 分析催化剂**：
- 哪些是有新闻或基本面主导的
- 哪些是没有任何利好消息就大幅波动的
- **没有新闻 + 没有基本面消息 → 上涨骗炮概率大**（"没有催化剂维持的上涨很难持久"）
- 通过 SEC 提交的文件了解：
  - 详细财务信息
  - 被市场忽视的投资机会
  - 重大事件：并购、重要合同、管理层变动、法律诉讼

**步骤3 — 进一步缩小范围**：
- 筛选那些涨幅大但**并不突出**的股票（避开热门股）
- 尤其是**开盘前已经有了一个巨量波段高点**
- 标记出这段走势的**关键阻力区域**
- 等价格再次到达关键阻力区附近 → 卖空

**避开**：太热门、交易量太大的股票（关注度高、流动性多 → 对手盘太强、容易被一波带走）
- 类比："酒吧里最漂亮的姑娘，很多人想搭讪 → 占便宜很难"
- 可以等第二天再狙击前一天的热门标的（已变配角）

## Exit & Stop Loss / 出场止损
**止损规则**（00:04:02-04:34）：
- **强制硬性止损 = 盘前波段的高点上方**
  - 如果价格能突破这个高点 → 可能继续上行
  - 必须硬性，不能犹豫，不能模糊
- **止损幅度约束**：单次最大损失不能超过"两天的平均盈利"
  - 例：日均赚 $100 → 单次最大损失不能超过 $200
  - 防止一次止损白做两周到一个月

视频未明确止盈规则，但案例显示 AMC 从 $70 做空到 $30 获利。

## Risk Management / 仓位
**信号评级 + 阶梯仓位**（00:04:35-05:22）：
- **A级信号**（过往胜率较高、共振多）：从平时 $10,000 放大到 **$100,000**
- **B级 / 共振较少**：正常 **$20,000 或 $10,000**
- **上限约束**：**最大交易头寸不能超过账户资金的20%**
- 类比 21 点：拿到20点也可能输给庄家21点，但拿到好牌不下注成不了优秀交易员

**止损强约束**：单次损失 ≤ 两天平均盈利

## Examples / 实例
- **AMC**：在 **$70 做空，$30 获利**，单笔约 **$700,000**（00:00:20）
- **实盘挑战**：2023年1月发起，35,000 → 100万美元用了55天（00:00:02）
- **第一次做空成功**：已退市的 VGG，入场后10分钟跌近50%，赚 $1000（00:01:37）

## Caveats / 注意
- 见好就收：自己规定**每年最多只赚一千万**（00:00:29）
- 19岁因女朋友嫌穷分手立志致富，卖掉汽车四个轮子凑了2000美元起家
- 前一年地狱期：只要买什么就崩盘（早期纯多头时期）
- 高人提醒可以做空 → 命运齿轮转动
- 阶梯仓位风险大，但"拿到好牌不下注也无法成为优秀交易员"
- 最大头寸20%上限是防爆账户的关键约束
- 每天只交易1小时，享受稳定盈利乐趣，不追求大目标
- "经常交易经常提现是最快乐的"
