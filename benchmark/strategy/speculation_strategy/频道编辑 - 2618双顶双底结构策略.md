# 2618 双顶双底中继结构策略 — 频道编辑

## TL;DR
2618 = "2" (双顶或双底两个相同位置) + "618" (跌破/突破颈线后的反弹/回落只到整个涨/跌幅的 0.618 斐波那契位)。是趋势中继结构：双顶出现在下跌趋势中作为下跌中继准确性高；双底出现在上涨趋势中作为上涨中继准确性高。趋势辨别常用均线 + 高高低低点。

## Setup / 形态

### 2618 双顶 (下跌中继) (05:50)
- **2** = 两个顶 (双顶)
- **618** = 价格走出双顶后跌破双顶颈线，之后跌势无法持续——先反弹到整个涨幅的斐波那契 0.618 后，再继续下行 (05:56)
- 适用：下跌趋势中作为 **下跌中继** (06:04) 准确性较高
- 上涨行情中作为 **逆势结构** 准确性差一些 (06:08)

### 2618 双底 (上涨中继) (06:13)
- **2** = 双底
- **618** = 价格走出双底后突破双底颈线，但涨势无法持续——先回到整个涨幅的斐波那契 0.618 后，再继续上行 (06:20)
- 出现在上涨趋势中 → 准确率会提高 (06:13)

### 趋势辨别 (06:11)
- 最常用的就是 **均线** 和 **高高低低点**。

## Entry Rules / 入场

### 双顶中继做空入场
1. 大趋势是下跌（均线 + 高高低低点确认）。
2. 出现双顶形态，价格跌破双顶颈线。
3. 反弹到下跌总幅度的斐波那契 0.618 位置遇阻。
4. 在 0.618 附近入场空单。

### 双底中继做多入场
1. 大趋势是上涨。
2. 出现双底形态，价格突破双底颈线。
3. 回调到上涨总幅度的斐波那契 0.618 位置获得支撑。
4. 在 0.618 附近入场多单。

## Exit & Stop Loss / 出场止损
- **止损**：双顶高点上方（做空）/ 双底低点下方（做多）。
- **止盈**：可参考前低/前高，或分批止盈。

## Caveats / 注意
- 顺势用——双顶 + 下跌趋势、双底 + 上涨趋势——逆势使用准确性下降。

- 0.618 是关键反弹/回落位，要求价格 "无法持续" 突破。
