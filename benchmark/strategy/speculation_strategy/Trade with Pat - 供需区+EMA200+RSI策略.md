# 供需区 + EMA200 + RSI 三共振策略 — Trade with Pat

## TL;DR
Trade with Pat宣称94.44%胜率的系统由"供需区域 + 4个指标 = 3组策略"组成，本策略是其中第一组：价格回到供给区/需求区附近 + EMA200趋势过滤 + RSI超买/超卖三个条件共振触发入场。视频作者BTC 30分钟图实测100次连续交易：1:1 胜率71%、1:2 胜率52%、按1:2操作累计收益93.9%——胜率赔率非常均衡。

## Setup / 形态

### 指标设置
- **EMA200** (00:55)
  - 在指标搜索栏输入 EMA
  - 参数 = 200
  - 颜色 = 白色
- **RSI** (01:01) (相对强弱指标 Relative Strength Index)
  - RSI = 黄色背景白色
  - RSI-based MA 勾掉
  - 阈值：70 上方超买，30 下方超卖
- **斐波那契回撤工具** (01:30) — 只留下三个参数 0 / 0.5 / 1（本子策略不使用，第三组策略才用）

### 供给/需求区识别 (01:47)
- 找供需区只需记住一个逻辑：供给和需求区是 **大资金机构** 的供给需求区——只有大资金才能推动大的价格运动；它们会在关键位置布置大量订单。
- 这些关键供需区域一般是行情启动的重要节点 (02:01)。
- 当价格再次回到这些节点时，大资金未成交的需求/供给订单就会阻止价格进一步行动。

### 四种供需区类型 (02:13)
- **需求区 = 跌平涨**：行情先跌、走平、再上涨；反转前的平台节点即需求区。
- **需求区 = 涨平涨**：行情先涨、走平、继续涨；上涨中继的平台节点。
- **供给区 = 涨平跌**：与需求区方向相反。
- **供给区 = 跌平跌**：上涨中暂停回落的中继。

## Entry Rules / 入场

### 多头规则 (03:18) — 三个条件全部满足
1. 价格回落到 **需求区** 附近。
2. 价格处在 **EMA200 均线上方或附近** (03:25)。
3. **RSI 在 30 线下方** 出现超卖信号 (03:30)。
4. 三条件满足 → 这根K线是关键K线，**收盘价入场**。

### 空头规则 (04:05) — 三个条件全部满足
1. 价格反弹到 **供给区** 附近。
2. 价格处在 **EMA200 均线下方或附近** (04:08)。
3. **RSI 在 70 线上方** 出现超买信号 (04:12)。
4. 三条件满足 → 关键K线收盘价入场。

## Exit & Stop Loss / 出场止损

### 止损 (03:38)
- **多头**：设置在 **需求区下方**。
- **特殊情况** (03:43)：如果关键K线的收盘价收在了需求区上方，但最低价跌破了需求区，那止损就设置在 **关键K线的下方**。
- **空头**：设置在 **供给区上方** (04:23)。

### 止盈 (03:50)
- 固定比例方式：1:1 / 1:2 / 1:3 等。
- 参考回测统计表格决定（视频作者测试结果显示 1:2 表现最优）。
- **止盈附加条件** (04:00)：
  - 多头止盈时，RSI 必须处在 50 中线 **上方**。
  - 空头止盈时，RSI 必须处在 50 中线 **下方** (04:28)。

## Risk Management / 仓位
- 视频作者建议在所做品种和周期上至少连续回测500次以上 (07:24)，对策略长期运行情况心中有数。
