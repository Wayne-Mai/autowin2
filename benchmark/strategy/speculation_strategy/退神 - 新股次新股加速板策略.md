# 新股/次新股巨量烂板次日加速板策略 — 退神（退学炒股）

## TL;DR
退神短线策略：新股/次新股连续大幅上涨后，前一天出现巨量烂板（放量滞涨），次日开盘走强超预期加速板/新龙头卡位时进场买入。万兴科技单笔赚 46%。10000 元两年做到千万。

## Setup / 形态
- 标的：A 股新股/次新股（加密货币同样适用）
- 周期：scalping/短线
- 战绩：
  - 2017.7.5 第一次资金翻倍
  - 8.28 从 10 万 → 20 万
  - 不到一个月 20 万 → 40 万
  - 11 月终止时 62 万
  - 2017.12.11 用 25 万本金重开
  - 2018.2.2 翻倍 → 3.8 117 万 → 4.2 302 万
  - 2019.4 突破千万

## Entry Rules / 入场

### 核心策略：新股/次新股加速板（02:30-03:42）

**触发条件**：
1. 新股或次新股
2. **连续大幅上涨后**
3. 前一天出现**巨量烂板**（放量滞涨）
4. 次日出现以下走势之一：
   - **开盘走强超预期加速板**
   - **新龙头卡位**
5. 进场买入

**逻辑**：
- 新股连续大幅拉升 → 庄家陆续出货 → 抛压持续增加
- 巨量烂板 = 放量滞涨 → 次日本应低开或持续走低
- 次日反而出现加速板 → 说明有足以吃掉当前抛压的需求入场
- 股价上涨速度超过市场预期 → 引发追涨心理 → 进一步推升

**新龙头卡位**（03:27-03:42）：
- 新股忽然成为某行业/板块的领头股
- 吸引更多投资者关注和买入
- 市场预期未来获得更好发展

**加密货币应用**（03:42-04:03）：
- 新币种有创新技术（账户抽象、去中心化身份等）
- 简单快捷成本更低的新技术 → 吸引资金进入板块
- 新币种成为某行业/板块龙头 → 价格更易被推高

### 实操选股原则（04:03-04:36）
- "我们要学会看别人赚钱"
- 几乎全部选当时最热板块下的强势股
- **顺势而为是信条**
- 与日本最强散户 CIS 不谋而合

### 经典案例（04:36-05:31）

**北京科锐（2017.5）**：
- 雄安概念股
- 一次涨停后第二天集合竞价时买入
- 看到涨停打开（市场需求强烈，买盘超卖盘） → 5.22 罕见加仓
- 转天获利离场

**津膜科技（2017.5）**：
- 雄安概念股
- 观察到股票一次涨停后出现放量并继续保持强势
- 在第二天顺势买入

其他案例：诚邦股份、日盈电子、海鸥股份等

### 突破策略（05:31-06:16）

**万兴科技（2018）**：
1. 价格连续几天达到当天允许最高限度（连续一次涨停）→ 需求强烈
2. 股价开始横盘调整
3. 3.1 股价再次上涨到当天最高限度
4. **在上升三角形态末期**
5. 突破后价格仍然保持强势
6. 退神 3.2 入场买入，持有到 3.8 离场
7. 46% 账户收益

## Exit & Stop Loss / 出场止损

**核心铁律**："止损是神的武器"（06:52）

**止损规则**（06:52-07:34）：
- "错了就要果断割肉"
- 一定不能抱有任何幻想
- 明知风险却妄想发生奇迹 = 股市中的残疾人
- 经常买在最高点也不影响赚钱，因为短线客即使入在高位也会从容退出
- "绝对不抽"

**实例**：
- 高争民爆：连续拉升后前一天出现巨量烂板，第二天高开买入，走势不如预期 → 果断卖出
- 华北高速、君禾股份等：发现股价未按预期运行立刻清仓，**无一例外**

**持仓与回撤线（砍仓手法）**（07:42-08:03）：
- 找到适合自己的持仓方式
- 设置一个**回撤线**
- 资金在回撤线之上 → 全仓
- 资金到了回撤线 → 分仓
- 回撤线根据资金最高点变动，**大概 10 个点左右的幅度**
- 既能保证攻击性又能防止大幅回撤

**止盈**：
- "到点就收"
- "从来不会放纵自己的贪婪"

## Risk Management / 仓位
- 容错空间充足 + 坚定执行止损是关键
- 回撤线动态管理仓位

- 必须坚定执行止损（不能抱幻想）
- 方式要与自己的性格相符
