# EMA50/EMA200 趋势震荡判断与移动止盈 — 频道编辑

## TL;DR
利用 EMA50 与 EMA200 的相对位置和距离动态判断市场处于趋势还是震荡：金叉/死叉后距离持续扩大 = 强趋势（可用作移动止盈基准）；两条均线距离持续缩小或上下缠绕 = 震荡盘整（不适合进入交易）(05:18-06:32)。

## Setup / 形态
- EMA50 + EMA200

## How To Read / 怎么解读

### 强多头趋势 (05:27-05:41)
- EMA50 由下向上穿过 EMA200 形成金叉
- 两条均线之间的距离逐渐增大

### 强空头趋势 (05:55-06:08)
- EMA50 由上向下跌破 EMA200 形成死叉
- 两条均线之间距离逐渐增大

### 震荡盘整 (06:08-06:24)
- EMA50 和 EMA200 之间的距离持续缩小
- 均线开始不断的上下缠绕 = 走势已进入无序的盘整期
- 不适合进入交易

## Trade Setups Using It / 配套的入场出场
- 主要用作**移动止盈**：已在底部入场后，观察价格已跌破 EMA50 → 全部止盈或部分止盈，留住利润 (05:43-05:53)
- 震荡判断：可作为停止交易、切换其它标的的信号

## Caveats / 注意
- 这不是独立的入场策略，是辅助判断与移动止盈工具
