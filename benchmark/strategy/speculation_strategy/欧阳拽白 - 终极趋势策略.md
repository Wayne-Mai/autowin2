# 终极趋势策略 — 欧阳拽白

## TL;DR
币圈最强趋势交易员欧阳拽白的核心策略：真突破时做多 + 市场情绪极度亢奋跌破均线时做瀑布短空。坚定执行简单的一买一卖，降低交易频次获得稳定收益。总仓位杠杆≤2倍。

## Setup / 形态
- 标的：比特币为主（币圈适用）
- 周期：日线为主（swing）
- 主张：每个市场都有独特形态；比特币市场最有用最简单的就是真突破做多 + 极端情绪做空（01:02-01:18）
- 长期主张："不抛币不套现不做空"

## Entry Rules / 入场

### 做多：真突破策略（01:22-02:24）

**三个核心条件**（必须全部满足）：
1. **识别趋势**：顺应趋势能有效增加突破是真的概率
2. **识别盘整区间**：盘整时间越长，突破位置越在整理形态末端，对真突破加成越大
3. **突破要放量**

**真突破示例**：
- BTC 日线长时间整理后在旗型末端放量突破 → 史诗级大牛市
- 另一例：盘整时间更长，突破后价格再也没回来过

**假突破特点**（02:08-02:24）：
- 在一段趋势的末端非常容易出现
- 价格突破后很快遇阻回落
- 突破的成交量不高
- 震荡的时间也比较短

### 做空：极端情绪瀑布短空（02:24-02:58）

**入场条件**：
1. 市场情绪极度乐观亢奋
2. 跌破均线
3. 均线参考：**MA30、EMA83** 等经常用的参考均线
4. 这是欧阳在币圈观察到最多的形态（与币圈散户多头杠杆率过高有关）

**市场情绪判断**：
- "牛市总是在绝望中来临，质疑中展开，犹豫中猛涨，狂欢中结束"
- 没人敢做空时、市场全部唱多时 → 流动性最好，逃顶出货最好的时机

### 市场情绪指标（03:00-04:32）

**持仓量**：
- 持仓量越大 → 人们持仓的兴趣越高
- 价格运动幅度越来越大，到关键区域附近时成交量很大，但持仓量开始断崖式下跌 → 一方大部分仓位已平仓或被强平 → 一场战役的结束
- ETH 回测：每次持仓量处在高位时，市场情绪极度亢奋 → 经常出现较大回调
- 持仓量经常会有一段时间的瓶颈区域，到达就下跌

**多空持仓人数比值**：
- 价格快速上涨但比值增长过快 → 散户追涨 → 下一个阻力位遇阻回落
- 异常配合：欧易多空持仓人数比值从 1 快速升到 2 甚至 3 → 容易出现极端下跌行情

**多头平均杠杆率**：
- 真实市场平均杠杆率很难超过两三倍（巨鲸杠杆率都很低）
- 观察各平台公开实盘交易员（数千名）的平均杠杆率作为参考
- 在关键位置参考非常有必要

## Exit & Stop Loss / 出场止损
- 视频未明确止损规则细节，但强调"低频的入场和低频的大止损可以有效提高胜率"（05:38）
- 短空：瀑布短线（短期出场）

## Risk Management / 仓位

**欧阳认为仓位和心态是这套交易系统中最重要的部分**（04:52）

**杠杆规则**（04:57-05:13）：
- **总仓位的杠杆一定不能超过 2 倍**
- 例：有 10000U，总的交易价值不能超过两万
- 对于不确定的行情：使用最大总仓位杠杆的一部分
- **金字塔式分批入场**
- 避免账户出现重大亏损
- 对实现稳定的低倍复利非常有帮助

**心态管理**（05:13-05:46）：
- 近期不断亏损 → 立刻停止交易，把高杠杆仓位全部平仓去休息
- 最近大幅盈利不能骄傲 → 立刻提现一部分利润
- "大多数愚蠢的错误都是在有盈利之后犯的"
- 交易最重要的是学会**等待**
- **低频入场 + 低频大止损 → 有效提高胜率**
- 最好**不要全职交易**，要在场外有自己的事业
- 持续不断的后备资金是平稳心态的定心石

## Examples / 实例
- BTC 3000 多美金时果断抄底做多，一单赚到几十万 USDT（00:22-00:29）
- 从几千美金做到几千万美金（00:11-00:14）
- 牛初开始一路喊不抛币不套现不做空

## Caveats / 注意
- 只做一边的好处：减少交易频率，不会被行情来回收割
- 持仓量"断崖式下跌" 通常是行情结束信号
- 必须配合后备资金（场外事业），全职交易容易心态崩
- 极端情绪做空依赖币圈散户杠杆率过高的特性
- 假突破频出 → 真突破三条件缺一不可
