# ICT日内流动性狩猎策略 — JadeCap

## TL;DR
JadeCap 标准三步流程：(1) 日线确定多空偏向 + 锁定前一日高低点为目标；(2) 在纽约时段早盘 (9:30-10:30 ET) 等待止损密集区被扫；(3) 切到5/15分钟，在反转K线收盘价/FVG/订单块入场 (00:38 整段)。

## Setup / 形态
- 股票、外汇、加密货币市场都有效 (00:35-00:38)
- 高质量可重复信号，每天提供入场机会
- 利用聪明钱"积累-操纵-派发"三阶段框架 (01:58-02:14)

## Entry Rules / 入场

### 第一步：日线方向偏向 (00:47-01:40)
- 交易前必须建立坚定多空判断
- 主要使用两个重要位置：**前一日的高点或低点**
- 高低点不特指24小时内，也可以是近期重要波段的高点/低点
- 只要市场正在逐渐向这个水平移动，它就可能成为目标
- 主要方向确定后不要轻易改变多空信念
- 如果打开图表时价格已经在朝预期方向大幅波动 → **小仓位或不交易** (盈亏比不再有利，胜率低)

### 第二步：标记时段流动性关键区 (01:40-02:49)
**三力合一原理**：积累 → 操纵 → 派发
- 小时间框架先形成积累区间
- 操纵价格去打掉流动性
- 然后向预定的高时间框架目标派发
- 95%的看涨K线最开始都是看跌；看跌K线最开始也大部分都是看涨
- 市场经常先反向波动以扫流动性，然后再转向

**标记常见的止损密集区/诱多诱空点**：
- 前一日的高低点
- 亚洲时段的高低点
- 伦敦时段的高低点
- 这些位置必须在**纽约时段破位**

**最佳交易时段** (02:39-02:48)：
- **美东时间 9:30-11:30 AM** 是交易机会高发期
- 止损单流动性被扫 = 潜在行情启动信号
- 所有日内低点/高点，大概 **70%** 都是在 **9:30-10:30 AM ET** 形成 (04:19-04:33)

### 第三步：5/15分钟入场 (02:49-03:33)
**三种入场方式**：
1. 扫掉日线高/低点流动性后的**反转K线收盘**入场
2. **公允价值缺口 FVG** 附近入场
3. **订单块**：价格大幅移动之前出现的最后一根看涨或看跌的反向K线 (JadeCap认为是聪明钱积累订单的区域)

**建议**：新手不要试图一次性学会所有入场模型；找一个最适合自己、最能理解的入场模型，然后大量实战练习。

## Exit & Stop Loss / 出场止损

**止损** (03:30-03:45)：
- 应设置在盈亏比合理位置
- 例如：**日内的高点上方或低点下方**
- 如果基于 FVG 入场：止损可以设置在 **FVG 的 50% 位置**，或缺口的顶部/底部附近

**主要止盈目标**：始终是第一步中确定的**前一日高/低点或更重要的波段高低点** (03:45-03:51)

**时间退出规则** (03:51-04:09)：
- 不一定非要等到价格完全触及预设日线目标才平仓
- 在**纽约交易时段的午间或下午**，日内最高/最低大概率已经形成
- 日内交易员不隔夜持仓 → 可以开始平掉仓位
- 波段交易员 → 参考更高周期的流动性区，结合价格行为与风险承受度

## Risk Management / 仓位
- 视频未给出具体仓位/百分比公式
- 主要约束是时段过滤：错过最佳时段不交易
- 主要风险点：错过最佳时段 + 价格已大幅波动 = 小仓位或不交易

## Examples / 实例
**纳指空单完整例子** (04:33-05:38)：
1. 日线处在短期下跌趋势 → 日线方向看空
2. 标记前一日高点作为潜在买方流动性区
3. 价格在纽约时段突破前一日高点 → 清扫了空头止损 + 吸引了突破买盘入场
4. 但行情没能延续上涨 → 这是一次流动性狩猎，不是真正强势突破
5. **入场机会 A**：价格突破前一日高点后没能继续上行 → 立即做空，止损设在流动性高点上方
6. **入场机会 B** (错过A时)：等价格下跌后形成 FVG，价格回补被拒绝时做空，止损同样放在流动性高点上方
7. 入场后价格继续下跌，**美东 12:30 PM 左右触及亚洲时段低点** → 在这里平仓
8. 正好与日内低点形成时机吻合，避免了午后震荡行情

## Caveats / 注意
- 一旦打开图表发现价格已经在朝预期方向大幅波动 → 小仓位或不交易
- 新手避免一次学会三种入场模型，选一种专精
- 必须等到纽约时段流动性区破位才执行入场
- 在最佳时段以外触及的高低点也可作为后续目标
- 95%的K线"先反向再正向"原则要求严格的纪律
