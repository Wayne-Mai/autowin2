# Money Flow Profile 资金流动概况策略 — Speculation Lab

## TL;DR
Money Flow Profile 是集合交易量、资金流动、市场情绪的综合指标。通过黄色区域（回溯期内资金活跃度最高的"公允价格"）+ 热力图 + 情绪图，精确定位止损位与入场点。

## What It Measures / 它衡量什么
- 不同价格水平的交易活动和资金流动分布
- 市场情绪（买家多 / 卖家多）
- 回溯期内资金活跃度最高的价格位置 = "公允价格"

## Parameters & Settings / 参数设置
- TradingView 指标搜索栏输入 **Money Flow Profile**
- 添加到图表
- 回溯期时间范围可修改
- 分布热力图可基于交易量 或 情绪 显示

## How To Read / 怎么解读

### 三个数据区（03:25-03:36）
- **右侧**：不同价格水平的交易活动和资金流动分布
- **左侧**：对应价格水平的市场情绪
- **中间**：各个价格水平的具体价格

### 黄色公允价格区域（03:41-04:06）
- 贯穿图表的黄色区域基于交易量计算
- 回溯期内买卖资金活跃度最高的位置
- 资金在这段时间最喜欢在此价格位置买卖
- 价格远离后**往往还会再回到这里**
- 经常让价格出现停止行为并保持在这附近
- 是这个时间范围活跃度最高的公允价格

### 热力图（04:06-04:17）
**基于交易量**：
- 红色区域 = 交易活动比较少
- 蓝色区域 = 交易很活跃

**切换到情绪图**：
- 红色 = 卖家多的位置
- 绿色 = 买家多的位置

### 最高情绪区功能（04:24-04:38）
- 显示买家或卖家最多的价格区间
- **对设置止损非常方便**
- 在不容易到达的位置设置止损
- 然后在附近寻找交易信号
- 盈亏比和交易计划就会非常容易确定

## Trade Setups Using It / 配套的入场出场
- 入场：在公允价格区域寻找停止行为信号
- 止损：放在"最高情绪区"之外（不易到达的位置）
- 止盈：以公允价格作为均值回归目标

## Caveats / 注意
- 回溯期设置影响公允价格位置
- 该指标是综合工具，建议与其他成交量指标（如 SVP、OBV）结合使用
