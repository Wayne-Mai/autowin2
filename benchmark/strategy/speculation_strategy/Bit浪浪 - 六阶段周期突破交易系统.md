# 六阶段周期突破交易系统 — Bit浪浪

## TL;DR
币圈现役最强短线交易员之一，将币种从冷到热划分六阶段，只在阶段1/2/4做顺势突破。胜率仅42%但通过严格分仓（每次固定仓位）保证盈利曲线无大回撤。还配合"春夏秋冬"市场温度理论控制总仓位与频率。

## Setup / 形态
**币种六阶段周期划分**（00:01:11-02:21）：

| 阶段 | 描述 |
|---|---|
| **阶段1**：行情启动 | 突破大级别震荡结构开始启动 |
| **阶段2**：小分歧 | 主升浪到达目标位（压力位/整数位）前的一个或多个小分歧 |
| **阶段3**：大分歧 | 主升浪目标位到达后的大分歧（震荡持续较长）— 难做逆势 |
| **阶段4**：二波拉升 | 大分歧结束后延续主升浪（中间也有分歧） |
| **阶段5**：见顶 | 行情到达最终目标位后见顶 — 难做逆势 |
| **阶段6**：末期下跌+震荡 | 行情进入末期下跌再到震荡 — 难做 |

**绝佳满仓阶段**：1 和 4（"任何时候都要满仓梭哈的位置"，00:02:48）
**次选**：阶段 2
**不建议**：阶段 3、5（逆势难判反转）、阶段 6（无规律）

## Entry Rules / 入场
**阶段1/2/4 交易逻辑**（00:02:54-04:42）：

**步骤1 — 确定大级别趋势和阶段**：
- 日线或小时线判断当前所处的阶段

**步骤2 — 判断上方盈利空间**：
- 距离价格最容易出现停止和反转的位置幅度有多大

**步骤3 — 切换小级别图表找"企稳拐点"**：
- 在15分钟图表找顺势的企稳拐点

**企稳拐点定义**（00:04:29-04:41）：
- 冲高回落的**第一波调整不能动**
- 反弹后也**不能动**
- **再次反弹后再下跌企稳，价格跌不下去的时候**才尝试做多
- 做的是"突破的结构"，不是"突破的点位"

**大小分歧的提前判断**（00:03:50-04:09）：
- 小级别图表到关键阻力位 → 容易出现**小分歧**
- 日线级别很关键或很高的位置 + 前面已经几段上涨 → 容易出现**大分歧**
- 大分歧后：短期不参与，等收敛到末端再尝试
- 震荡时间越长越好，收敛幅度越小越好

**People 实例**（00:03:12-03:38）：
- 一段下跌震荡后，一根17%大阳线快速突破震荡区间 = 阶段1
- 没赶上 → 等出现分歧结构 = 阶段2
- 5月20日在阶段2分歧结构位置入场
- 上方关键阻力：0.07 → 0.085

## Risk Management / 仓位
**分仓法（核心）**（00:05:17-05:55）：
- 例：3万美元资金 → 分成3份，每份1万
- **每次只用其中一份开仓**
- 亏了1000 → 再补进1000（保持1万）
- 赚了1000 → 把1000提走（保持1万）
- 确保每次开仓**固定金额**
- 直到能用3万赚到6万 → 才把每次开仓提高到1.5万或2万
- **个人风险承受上限不同**：心脏不能承受就不要加仓

**分仓两个好处**：
1. 即使遇见交易所插针/极端行情，最多爆掉1/3资金（强制第二止损）
2. 固定仓位 → 亏损或盈利时心态都更平和

**其他约束**：
- 设止损
- 低倍杠杆
- 不逆势交易

**胜率42%、无大回撤** = 完全靠分仓 + 大盈亏比

## Caveats / 注意
- **春夏秋冬市场温度**（00:06:38-09:03）：
  - **春天**（行情启动）：只有少量资金预判
  - **夏天**（鼎盛狂热）：山寨币最猛（ORDI/SOL等冒头），爆发出现，可激进
  - **秋天**（高点回调后震荡）：补涨币像秋后的蚂蚱，但反复止损
  - **冬天**（市场熄火）：选择冬眠
  - 秋冬：止损/杠杆全降低
  - 春夏：相对激进
- **板块龙头规律**："不从初始跟涨过的不可能是龙头"
- 比特币是风向标，肉眼看到BTC突破重要位置在拉升 + 市场够热 → 找板块龙头
- 阶段3和5都有多头进场，很难确定真正拐头时机，容易被反弹/插针打损
- 阶段6波动无规律
- "在阶段一和四之外的任何一笔交易都属于犯病"（00:01:25）
