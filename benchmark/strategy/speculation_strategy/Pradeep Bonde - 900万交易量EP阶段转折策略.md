# 900万交易量EP阶段转折策略 — Pradeep Bonde

## TL;DR
EP = "Earnings/Episodic Pivot" 阶段转折，核心是"由于催化剂，市场改变了对标的的看法，引导价格朝不同方向发展"。Pradeep Bonde 在此基础上加了"900万股以上交易量突破"硬条件——这种极端高成交量突破通常带来巨大后续波动。配合 MAGNA 五三 + CAP10×10 选股清单筛选高质量标的，是其门徒（Qullamaggie、Kelvin Wong等）做出亿万美金账户的核心策略。

## Setup / 形态
- 全称: "900万股交易量EP策略" (01:11)
- 关键特性 (00:54)：催化剂导致市场对标的短期或长期发生完全不同的评估，引导价格朝不同方向。
- 硬条件 (01:14)：股票必须在催化剂作用下短时间内出现 **900万股以上** 的交易量突破。
- 极端高成交量突破通常带来巨大波动 (01:20)；有些股票一年内会反复出现900万股突破，每次都伴大波动——Pradeep Bonde称之为"糖宝宝"，加入观察列表反复观察 (01:42)。

## Entry Rules / 入场

### MAGNA五三 + CAP10×10 选股清单 (01:50)

**MAGNA (必选项)**：

- **M = Massive Acceleration** (02:03)：业绩爆发式增长。普遍预估10%但实际公布100%、200%甚至300%的巨大增长。这种大规模超预期会推动价格大规模变动。
- **G = Gap Up** (02:23)：股票必须在盘前大幅跳空或急剧上涨。催化剂足够意外则市场应有缺口；但跳空/急涨不能过大要适中。反例 ASPN (02:45)：销售额突然大增但第一天涨跳过大，之后只走20%——催化剂在第一天就被消化掉。
- **N = Neglect (忽视)** (02:58)：尽量寻找被市场长期忽视、价格长期低迷的股票。过去几个月甚至一年都没有过大涨；很少机构介入、很少人讨论、新闻都没有。示例 Root (03:15)：长期被忽视，财务数据爆发后引发一轮大行情。
- **A = Sales Acceleration** (03:24)：销售额加速不是依赖补贴等财务因素，而是核心业务增长、需求增加导致；不像裁员、资产剥离等不持续性因素。能反映基本面良好，大资金机构更愿入场。

**五三 (可选项 / 加强)** (03:44)：

- **5** (03:48)：下跌卖空时间应持续5天或更长。此时价格突然上涨，空头会恐慌性平仓引发空头挤压，导致价格上涨加速。
- **3** (03:59)：至少3名以上分析师上调合理价格预估。基本面突然改变时，分析师通常会提价。示例 SMCI (04:09)：利好后12家分析师立即调高合理价格预估。

**CAP10×10 (爆炸性走势上限条件)** (04:14)：

- **市值<100亿美元** (04:18)：更容易翻倍或翻几倍；百亿千亿市值即便有催化剂短期涨幅也有限。
- **IPO时间<10年** (04:35)：大部分股票最大涨幅基本都发生在IPO后前10年，尤其前5年涨幅最大。可用 briefing 这类网站查询。

### 入场点 (04:48)
满足 MAGNA五三 + CAP10×10：
- **强催化剂**：直接在开盘价买入 (04:57)。
- **不确定催化剂强度**：观察市场反应再决定是否进场 (05:00)。
- **标的涨幅过大**：延迟等待回调后再入场 (05:07)。

## Exit & Stop Loss / 出场止损
- **强催化剂止损**：10% (05:13)
- **一般催化剂止损**：2.5% (05:17)
- **失效退出** (05:19)：股价突破关键点位但没能持续上涨，迅速退场并重新评估。
- **预设盈利目标** (05:25)：入场前提前设定客观合理的盈利目标，比如预计涨40-50%，那么涨到40%后开始逐步减仓 (05:33)。
- **表现不符预期** (05:37)：立即退出。
- **走势犹豫** (05:45)：通常表现最好的股票入场后几乎不会回头，如果走势犹豫不决也应尽早离场。

## Risk Management / 仓位
- 风险承受能力直接决定回报幅度 (00:32)。
- 强催化剂可承受10%止损 vs 普通催化剂只能2.5%，相当于按催化剂强度匹配仓位/止损宽度。

## Examples / 实例
- ASPN (02:45)：销售额突增但第一天跳空过大→后续只涨20%（反例）
- Root (03:15)：长期被忽视+财务爆发→大行情（正例）
- SMCI (04:09)：12家分析师调高目标价（"3"的范例）

## Caveats / 注意
- MAGNA必选、五三可选；缺少MAGNA任何一项都不应交易。
- 缺口必须有但不能过大——过大说明催化剂已被消化。
- 平时积累"糖宝宝"观察列表 (01:46)：定期复盘那些反复出现900万股突破并伴大波动的标的，长期持续追踪。
- 表现最好的股票几乎不回头，犹豫不决就是信号 (05:43)。
