# 25日均线乖离率逆向交易策略 — BNF（小手川隆）

## TL;DR
"日本交易之神"BNF的早期成名策略。在熊市中通过 25日移动平均线乖离率识别被严重低估的股票（负值乖离率大），在反弹中获取利润。21岁本金 160万日元 → 三年做到 1亿日元，全靠这套逆向投资。

## Setup / 形态
- **市场环境**：熊市 / 互联网泡沫破裂期（2000-2003）
- **品种**：日本股票
- **持有期**：一天到一周不等的**短线交易**（00:03:02-03:06）
- **核心理念**：约翰邓普顿："行情总在绝望中诞生" — 市场短期内常存在过度反应，资产价格偏离基本面 → 寻找被低估的资产，低位买进 → 价格回调到合理水平时获利

## Entry Rules / 入场
（00:04:09-05:13）

**核心指标 — 25日移动平均线乖离率（偏离率）**：
- **乖离率 = (当前价格 − 25日均线) / 25日均线 × 100%**
- 假设25日均线 = 100日元，当前价 80日元 → 乖离率 −20%
- **大幅负值乖离率 = 价格被严重低估，可买入博反弹**
- 正值过大 = 价格被短期高估，警惕回调

**入场条件**：
- 找乖离率为**显著负值**的股票
- 不同的股票/行业的乖离率基准不同
- BNF对大型股、小型股、不同行业设定各自的基准（详见"实例"段，视频画面有"当时他总结的入场参考数值表"，但具体数值未在转录中给出）

**纪律约束**（00:05:04-05:19）：
- 在初始资金积累阶段：
  - **只做现货交易**
  - **拒绝做空和杠杆交易**
- 理由：杠杆增加投资亏损风险，可能陷入更深困境
- "控制风险比扩大收益更重要"

## Exit & Stop Loss / 出场止损
- 价格反弹回到合理水平时获利
- 短线持有：一天到一周
- 视频未具体讲止损规则

## Risk Management / 仓位
- 只做现货
- 拒绝杠杆和做空
- 短线持有控制风险

## Examples / 实例
- 2000-2003 日本熊市环境
- BNF 用 160万日元（约21岁，2000年）开始
- 半年内长线投资损失一半本金（互联网泡沫破裂期）
- 转为短线逆向投资后三年做到 1亿日元

## Caveats / 注意
- 逆向投资"根据市场偏见赚钱，是一个难度很高的游戏"
- 需要：
  - 保持乐观主义精神
  - 不跟风操作
  - 付出大量时间研究
- "贸然抄底一有不慎就会被拖入地狱"（00:04:04）
- 2003年起这套方法因为日本股市转牛，乖离率逆向法**变得越来越不起作用**（00:05:22-05:34）
  - "市场行情不好时，逆势可以捡便宜买进；行情变好时容易错失买入机会"
- 这是BNF分水岭之前的策略，后转向顺势策略（见 BNF顺势策略.md）
