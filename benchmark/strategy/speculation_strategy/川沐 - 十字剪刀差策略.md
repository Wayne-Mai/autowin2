# 十字剪刀差策略 — 川沐

## TL;DR
通过观察山寨币"合约持仓量"与"合约持仓名义价值"两条曲线，捕捉庄家在拉盘过程中悄悄平多换空的异动信号。当价格还在上涨但持仓量与名义价值出现明显开口（剪刀差）时，就是庄家换方向的强信号，逆势做空大概率会有惊人收益。

## Setup / 形态
- **核心标的**：持续爆拉的山寨币（00:03:44）
- **前置条件 — 强庄信号**：
  - 合约持仓量占市值的比例高 = 多空对手盘充足
  - 持仓量甚至超过市值 = 庄家筹码越足，越有拉盘动力（00:03:54-04:07）
- **正常拉盘特征**（00:04:07-04:13）：
  - 价格上涨，合约持仓量持续增加
  - 合约持仓的"名义价值"与持仓量保持一致，**持续贴合增长**

## Entry Rules / 入场
**信号出现条件 — 十字剪刀差**（00:04:13-04:55）：
1. 价格还在上涨（甚至持续创高）
2. 合约持仓的**名义价值与持仓量突然分离**，出现明显的"开口空洞"
3. 具体表现：**持仓量在跌，名义价值在涨**
4. 含义：庄家通过拉盘或托盘平多，在卖出多头仓位并建立空单
5. 信号出现 → 跟随庄家做空（逆势）

**信号源**：交易所/数据平台的合约持仓量与名义价值曲线对比。

## Exit & Stop Loss / 出场止损
- 视频未明确止损规则，但案例显示标的可暴跌至原价的5-10%
- 暗示：找到强信号后用低杠杆持有，等价格充分下跌
- 推荐做法：参考批量空单的止损止盈（涨40%止损 / 跌50%止盈）作为基础

## Risk Management / 仓位
- 川沐说"哪怕是开了1000美金的空单，直接都可以退休了"（00:05:07）— 强调该策略一旦命中收益极大
- 显示该策略小仓位即可博取大收益，不应重仓博

## Examples / 实例
1. **OM**（00:04:55-05:10）：
   - 2025-03-26 出现十字剪刀差信号时，价格仍在6.5以上

   - 跌幅约94%，开1000美金空单可"退休"

## Caveats / 注意
- 必须同时具备"价格上涨 + 持仓量下跌 + 名义价值上涨"三个分离条件
- 仅适用于强庄山寨币（持仓量/市值比例高）
- 数据信号需从专业数据面板获取（持仓量、名义价值）
- 庄家在剪刀差出现后还可能继续拉一段才崩盘，止损位置要给足空间
- 是逆势交易，需要识别"庄家换方向"而非追跌
