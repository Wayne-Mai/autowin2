# Volume Profile / VPVR 成交量分布图 — 频道编辑

## TL;DR
Volume Profile（成交量分布图）是 TradingView 公认最强的量能指标——是唯一被单独列为一组的指标族（共5个）。它把传统纵向"时间维度成交量"翻转成 **横向"价格维度成交量"**——直观告诉交易员：其他交易者喜欢在哪些价位交易、哪个价格被多空所接受、公认的有交易价值的区间在哪里。任何策略结合 Volume Profile 都能大幅提升胜率。

## What It Measures / 它衡量什么

### Volume vs Volume Profile 本质区别 (01:11)
- **Volume (传统成交量)**：表示在一个 **特定时间周期** 内的成交量。日线图 = 每天总成交量；小时图 = 每小时成交量。是 **竖向** 量能柱。
- **Volume Profile (成交量分布图)**：表示在一个 **特定价格范围** 内的成交量。是 **横向** 量能柱。

### 核心信息
- 其他交易者喜欢在哪些价位交易 (01:33)
- 哪个价格最被多空所接受
- 大家公认的有交易价值的区间在哪里

### 关键概念
- **POC (Point of Control) 控制点** (01:51)：成交量最大的价格水平（红线标记）。由于交投活跃，价格再次到此位置通常很难一次突破——常被视为 **支撑/阻力位**。
- **VAH (Value Area High)** + **VAL (Value Area Low)** (05:09)：公认有价值密集成交区的最高价格和最低价格——形成区间边界。
- **Value Area (数值区)**：以POC为中心、占总范围70%(默认)的高交易量区域。

## Parameters & Settings / 参数设置

### 一组5个指标 (00:46)

**1. Fixed Range Volume Profile (固定范围成交量分布图)** (00:52)
- 点击指标 → 选中范围的起点和终点 → 显示该范围的详细成交量分布
- 适合观察某段单边行情、某个箱体或持续震荡的走势

**2. Periodic Volume Profile (周期成交量分布图)** (06:33)
- 可观察一根K线的成交量分布，类似订单流
- **阶段数值**：若改成7 = 可观察7根K线范围的成交量分布
- **Bar/Day切换**：改 Day = 观察7天内的成交量分布

**3. Session Volume Profile** (06:55)
- 每一天的成交量分布情况
- 判断趋势和供给/需求区位置

**4. Session Volume Profile HD (高清成交量分布图)** (08:09)
- 显示非常精细，几乎每个价格上的成交量都一目了然
- 适合日内交易者、价格行为研究者、按分钟按秒交易者、做市商

**5. Visible Range Volume Profile (VPVR)** (08:37)
- 显示当前屏幕可见K线范围内的所有量能分布
- 图表变动时VPVR自动调整
- 功能与其他4个一样但使用更方便、图表更简洁

### 固定范围 Volume Profile 设置栏 (04:16)
- **行数 (Rows)**：量能柱的行数，越大量能柱越密集（默认24，可设100更细化）。
- **Value Area Volume**：默认70——突出显示以POC为中心占总范围70%成交量的区域，剩余30%颜色变暗。
  - 改成100 = 所有成交量颜色都变得非常清晰
  - 改到20以下 = 只看成交量最密集区域，其它变暗（识别真正有价值的密集成交区） (04:48)
- **样式 Up Volume / Down Volume**：设置暗色区域颜色。
- **Value Area Up / Down**：设置高量区域颜色。
- **VAH / VAL 选项**：打开 → 区间会有边界。
- **数值选项**：在每根蓝/黄色量能柱上显示买入和卖出的数值。
- **宽度百分比** (05:26)：默认30——量能柱宽度占范围30%。改100则POC量能柱扩展到范围右边界（美观用，无特殊意义）。
- **配置位置**：默认"左"；可选"右" → 量能柱在范围右侧显示。
- **发展控制点POC** (05:46)：显示该范围内POC的发展轨迹（之前的控制点也很重要，当价格再次接近会构成停止行为）。
- **发展数值区 VA** (06:09)：以POC为中心的有价值区域的发展情况，一般用不到。
- **Histogram Box** (06:18)：确定成交量分布图区域的背景颜色和不透明度。

## How To Read / 怎么解读

### 1. POC 作为支撑阻力 (02:03)
- POC红线所在价位 = 成交量最多 → 价格再次接近时通常很难一次突破 → 视为支撑/阻力位。
- 若POC与斐波那契、供给需求区等位置重合 → 确认这些阻力有效。
- 价格再次接近重合阻力 → 反转概率增加 (02:43)。

### 2. POC 趋势判断 (Session Volume Profile) (07:09)
- **下跌趋势的需求区识别** (07:13)：
  - 每天POC持续下降 = 每天有很多人认为价格还会跌得更低、卖筹码
  - 直到某天POC开始抬升 → 没有出现更低的POC → 越来越多人认为当前价格合适买入
  - 这个区间就是真正的需求区 (07:36)
  - **连续降低一周以上 + 下跌幅度越大 → 需求区的支撑效果越强** (07:43)
- **上涨趋势的供给区识别** (07:52)：
  - POC连续抬升一周以上 → 第一次出现下落
  - 下落的POC与顶部POC形成的区间 = 供给区
  - 价格再度到达 → 遇阻回落概率非常大 (08:05)

### 3. 密集成交区 + 斐波那契共振案例 (02:46)
- BTC 2021年11月-2022年1月 第一波下跌行情：
  - 密集成交区在 45,700-49,500 美元
  - POC在 49,000 美元附近
  - 斐波那契 0.382 回撤位置与密集成交区重合 → 形成阻力区间
  - 价格遇阻回落 → 第二波下跌开始 (03:19)
- 第二波下跌：底部 17,592 美元，POC 3万美元 → 斐波那契 0.382 又与密集成交区重合 (03:35)

## Trade Setups Using It / 配套的入场出场

### 反转入场场景
- 价格接近 POC / VAH / VAL / 密集成交区时，结合：
  - **MACD 连续顶/底背离** (03:53)
  - **Vegas 通道阻力共振** (04:01)
  - **K线形态**（头肩顶、双顶、吞没等）
  - 多信号验证可极大提高交易确定性 (04:10)

### 趋势中继
- POC 持续上行 → 上涨趋势中的成交合力
- POC 第一次下落 + 与历史POC形成区间 → 供给区做空

## Caveats / 注意
- 是 **确认指标**，需要和其他指标相互印证（斐波那契、MACD、Vegas、K线形态等）。
- POC 红线位置是核心，但 VAH/VAL 也同样重要。
- 不同 Volume Profile 变体适合不同场景：固定范围/某段行情、周期/某根K线、Session/每日、HD/精细日内、VPVR/可见全局。
- 单独的 POC 触碰不构成交易信号 —— 需要其他信号共振。
