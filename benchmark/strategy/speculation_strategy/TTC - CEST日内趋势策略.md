# CEST 日内趋势策略 — TTC (The Trading Channel)

## TL;DR
拥有200万跟随者的顶级交易员TTC 的"印钞机"策略——10年交易生涯中用过的最好的策略。CEST = Conditions (大周期EMA50+高高低低点判趋势) + Entry (小周期双顶双底/头肩顶底突破入场) + Stop (重要支撑阻力) + Targets (分批/1:2固定)。简单到可作"下单前检查清单"。BTC 4H+15M测试1804个交易日、1:2胜率49.4%、总收益80.12%。

## Setup / 形态

### 核心思路
- 顺大逆小的趋势策略 (05:03)。
- 参考 **大** 和 **小** 两个时间周期 (00:43)。
- 大周期判方向，小周期找入场结构。
- **整合期或趋势不清晰时不交易** (00:54)。

### 大周期判趋势 (00:48)
- **EMA50** (50日均线) + **连续高高低低点**
- 设置：指标搜索 EMA → 收藏 → 添加图表；**长度50** + 颜色白色。
- 比特币示例：用 **4小时图** 判大周期趋势 (01:03)。
- 上升趋势条件 (01:15)：
  - 价格处在 EMA50 上方
  - 连续出现更高的高点和更高的低点
  - 两条件满足 → 上升趋势 → 寻找买入机会
- 下跌趋势条件 (01:35)：
  - 价格处在 EMA50 下方
  - 连续出现更低的高点和更低的低点
  - 两条件满足 → 下跌趋势 → 寻找卖出机会

### 入场结构 (小周期 15分钟)
- 两种结构：**双顶/双底** 或 **头肩顶/头肩底** (02:03)

## Entry Rules / 入场

### 三步流程

**第一步 — 判方向**：4小时图 EMA50 + 高高低低点确定多/空方向 (00:46)。

**第二步 — 找入场结构**：切换到 15分钟图寻找双顶/双底或头肩顶/头肩底结构 (01:57)：
- 价格突破颈线（多头）或跌破颈线（空头）→ 在 K 线收盘价入场 (02:12)。

**第三步 — 入场细化** (顺大逆小的进入位置)：
- 上升趋势：在更高的高点被突破后、再回调到 **前一个波峰附近** 时找机会顺势买入 (01:28)。
- 下跌趋势：在更低的低点被跌破后、再反弹到 **前一个波谷附近** 时找机会顺势卖出 (01:51)。

### 实例 — 双底做多 (02:14)
- 当前一段走势走出明显双底结构 → 价格突破颈线 → 在K线收盘价入场多单。

### 实例 — 头肩顶做空 (02:24)
- 4小时图：价格处在 EMA50 下方 + 连续更低高低点 → 大级别趋势向下 → 等反弹做空机会 (02:37)。
- 切到15分钟图：反弹中出现明显头肩顶结构 → K线收盘价跌破颈线 → K线收盘价入场空单 (02:50)。

## Exit & Stop Loss / 出场止损

### 止损 (03:01)
- 设置在重要的支撑阻力位、供给需求区、前高前低、双顶双底/头肩顶底的末端附近等。

### 止盈
- 分批止盈，或固定 1:2 风险回报比方式 (03:11)。
- 可参考同样的关键位置作目标。

## Risk Management / 仓位

### CEST 缩写定义 (03:20)
- **C = Conditions** (入场条件)：大时间周期EMA50 + 高高低低点判断
- **E = Entry** (入场模式)：双顶双底 / 头肩顶底
- **S = Stop** (止损)：重要支撑阻力位
- **T = Targets** (止盈目标)

### 核心信念 (03:53)
专业交易员都会通过大量交易的统计数据去了解和验证一套策略，并且很清楚其中亏损和盈利都是策略的一部分以保证对策略的信心。验证策略是否值得信任，最快的方式就是回测；机会样本不能少于100次。

## Caveats / 注意
- **整合期/趋势不清晰不能交易** (00:54) ——这是硬规定。
- 双顶双底/头肩顶底必须是 **明确** 的结构 + 收盘价 **突破/跌破颈线** 才算入场。
- 主播评分 6 分；这是顺大逆小的趋势策略，比较容易吃到大趋势，可搭配斐波那契、MACD 等指标共振大幅提升胜率 (05:12)。
- 不明白共振信号的话可回看供需交易和价格行为相关内容。
