# 孕线（母子线 / Inside Bar）突破策略 — 频道编辑

## TL;DR
孕线 = 市场方向尚未选择时的震荡状态。日线发现孕线后切换到5/15分钟会看到震荡走势——本质是做"标准的震荡突破"：跌破震荡区间下沿做空、突破上沿做多。双孕线（连续两个内含bar）盈亏比比单孕线更高，因为区间内多空换手更充分，突破后波幅更大。但孕线形态出现频率很高，并不是每个都值得参与，必须结合 EMA13/EMA55 趋势过滤。

## Setup / 形态

### 核心概念 (02:50)
- 孕线是市场方向尚未选择时的震荡状态。
- 日线发现孕线后切换到5分钟/15分钟小级别图表，会看到一段震荡走势 (02:55)。
- 运线交易其实是在做"标准的震荡突破" (03:01)：
  - 价格跌破震荡区间下沿 → 做空
  - 突破震荡区间上沿 → 做多

### 双孕线 vs 单孕线 (03:09)
- 双孕线的盈亏比要更高——因为价格在一个区间进行多空换手越充分，突破后的波幅一般也就越大。

### 指标过滤 (03:21)
- **EMA13 + EMA55** 两条指数移动平均线
- 多头趋势判定：EMA13 在 EMA55 上方

## Entry Rules / 入场

### 多头规则 (孕线在波段高点上方出现) (03:31)
1. EMA13 在 EMA55 上方（多头趋势）。
2. 孕线在波段高点的上方出现。
3. 母线 (mother bar) 形态要求 (03:33)：
   - 母线的上引线不能太长
   - 母线实体至少是子线的一倍以上
   - 子线要处在母线的上半部分
4. **入场触发**：第三根K线突破母线最高价时入场多单 (03:39)。

### 双孕线情况 (03:43)
- 如果第三根K线没有突破母线最高价、也没有跌破子线的最低价——而是形成了双孕线 (即第三根K线本身又是子线内的内含bar)：
- 入场价仍设置在 **母线的最高价位置**。
- 止损则改为设置在 **第三根K线的最低价附近** (03:52)。

### 空头规则
- 视频未单独详述空头规则但根据多空对称：EMA13 在 EMA55 下方 + 孕线在波段低点下方出现 + 母线实体至少子线1倍以上 + 子线在母线下半部分 → 第三根K线跌破母线最低价入场空单。

## Exit & Stop Loss / 出场止损
- **标准止损**：母线最低价附近（多头）或母线最高价（空头）。
- **双孕线止损**：第三根K线的最低价/最高价附近 (03:52)。

## Caveats / 注意
- 孕线出现频率很高，不是每个都值得做，必须用 EMA13/EMA55 趋势过滤 + 位置要求（在波段高/低点附近）。
- 母线上引线不能太长——避免上影线过长意味母线本身已被卖压压制。
- 子线必须在母线的上半部分（多）或下半部分（空），保持方向一致性。
- 双孕线盈亏比更高但出现概率较低。
