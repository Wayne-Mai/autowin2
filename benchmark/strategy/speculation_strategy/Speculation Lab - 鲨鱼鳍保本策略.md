# 鲨鱼鳍保本策略 — Speculation Lab（介绍 OKX 结构化产品）

## TL;DR
OKX 鲨鱼鳍是一种基于期权组合的结构化产品，本金以 USDT 计价（不承担币种波动风险），无论涨跌都有 3.5% 保底年化收益，若到期价格落在障碍区间内则可获取 4%-9% 浮动收益。3-7 天周期。

## What It Measures / 它衡量什么
- 一种**期权组合**衍生品，从期货/期权等衍生品市场赚取收益
- 收益曲线像鲨鱼鳍 → 命名
- 本质：用稳定币 USDT 申购，到期赎回 USDT + 收益
- 不承担另一币种涨跌波动风险

## How To Read / 怎么解读

### 看涨比特币计算（01:13-02:47）
**示例**：2023 年 7 月 27 日申购 10000 USDT 看涨 BTC，一周期限，年化收益率在 3.5% 到 9% 之间浮动。

**情况 1：到期价低于最低障碍价 29500U**
- 例：BTC 到期价 28000U
- 收益率 = 保底 3.5%
- 到期收益 = 10000 × 3.5% × (7/365) = **6.71 USDT**

**情况 2：到期价在障碍区间内**（最低 29500U / 最高 33800U）
- 公式：APY = (到期价 - 最低障碍价) / (最高障碍价 - 最低障碍价) × (最高收益率 - 最低收益率) + 最低年化收益率
- 例：到期价 31220U
- APY = (31220 - 29500) / (33800 - 29500) × (9% - 4%) + 4% = 6%
- 到期收益 = 10000 × 6% × (7/365) ≈ **11.5 USDT**

**情况 3：到期价超过最高障碍价 33800U**
- 例：BTC 到期价 35000U
- APY = 保底 3.5%
- 收益 = 6.71 USDT（与情况 1 相同）

### 看跌比特币计算（02:47-04:10）
算法与看涨相反，预设区间例如 24900U-29300U。

**区间外（< 24900U 或 > 29300U）**：APY = 3.5%，收益 6.71 USDT

**区间内**：收益率随币价升高而降低
- 公式：APY = 最高年化收益率 - (到期币价 - 最低障碍价) / (最高 - 最低) × (最高 - 最低收益率)
- 例：到期价 25780U
- APY = 9% - (25780 - 24900) / (29300 - 24900) × (9% - 4%) = 8%
- 到期总收益 = 10000 × 8% × (7/365) = **15.34 USDT**

## Trade Setups Using It / 配套的入场出场
- 闲置 USDT 放进鲨鱼鳍赚外快
- 持续复投相当于持续吃 3.5%-9% 的利息
- **双边对冲玩法**（05:11-05:23）：同时购买涨跌双边产品 → 既能获得基础收益，也有更高几率获得增益收益率
- 行情判断准确度高 → 在基础收益上获得更多利润

## Caveats / 注意
- **不支持提前赎回**（05:23-05:32）：确保投入资金是闲置的，否则行情来了想抄底动不了
- 7 天期复投时间紧：上一轮资金到账后距离新申购截止只有 3.5 小时，必要时设置复投提醒
- 收益率 3.5%-9% 看起来低，但是相对无风险
- 不承担币种波动风险（USDT 申购 USDT 赎回）
