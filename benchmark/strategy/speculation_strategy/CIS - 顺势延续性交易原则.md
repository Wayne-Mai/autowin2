# 顺势延续性交易原则 — CIS

## TL;DR
日本"最强散户"CIS没有具体方法，但提出了清晰的顺势原则：连续上涨的股票大概率继续涨、连续下跌的大概率继续跌。市场具有延续性，强者更强，弱者更弱。要接受市场力量，不与之对抗。CIS 的原则是 BNF 顺势策略的最佳补充。

## Setup / 形态
- **核心认知**（00:06:42-06:57）：
  - 大部分时候，连续上涨的股票大概率会继续涨
  - 持续下跌的股票大概率会继续跌
  - 这是 CIS 顺势的大前提

## Entry Rules / 入场
**1. 拥抱趋势延续性**（00:06:57-07:25）：
- 绝大多数人把上升和下降想象成五五开的概率游戏
- 但市场具有很强的**延续性**
- 强者更强，弱者更弱
- "我们要接受市场的力量，而不是与他对抗"

**2. 避免"逢跌买入"思维**（00:07:25-07:46）：
- "逢跌买入"想法应尽量避免
- 股票大幅上涨时害怕被套高点，等待短暂下跌再买
- 没人知道这个时机会不会出现
- 强牛市可能错过整个行情

**3. 避免"亏损加仓"**（00:07:46-08:05）：
- 买入后股价开始下跌 → 最好的做法是承认失败迅速退出止损
- 亏损加仓 = 已经失败的单子还要增加赌注 → 只会让损失越来越大

## Exit & Stop Loss / 出场止损
- 反向走 → 迅速止损退出
- 不要扛单、不要加仓博反弹
- 接受失败

## Risk Management / 仓位
**核心理念**（00:08:05-08:21）：
- "不要过度纠结胜率，重要的是账户的整体收益"
- 风险和损失是必然
- 不是规避失败，而是**及时止损**
- 不是不能亏钱，而是**尽量做到小亏大赚**

## Examples / 实例
- 2005年 J-COM 误下单事件，CIS 赚 6亿日元
- 同事件 BNF 赚 20亿日元
- 多年朋友，两人都是日股顶尖交易员

## Caveats / 注意
- **不要迷信金科玉律**（00:08:21-08:35）：
  - "市场是一个复杂动态的系统"
  - "所谓的规则一旦被广泛传播，往往会迅速失效"
  - "交易需要的是独到的视角和敏锐的判断力"
- **重大波动期是机会**（00:08:35-08:56）：
  - "真正好的交易员，往往诞生于重大股灾、经济危机或市场转折中"
  - "波动越大，隐藏的机会就越多"
  - 能冷静应对并果断行动的少数人脱颖而出
- CIS 没有给具体的入场指标 / 周期 / 数值 — 主要是认知层面的原则
- 与 BNF 顺势策略配合使用
