# Vegas隧道震荡区间网格策略 — 频道编辑

## TL;DR
市场99%的时间在震荡，1%的时间在波动。本策略以Vegas隧道（EMA144/EMA169）为震荡中轴，识别经历短期大幅波动后形成的对称性震荡区间，结合大周期趋势顺势分批入场，并加入MACD背离与K线形态作为信号共振。

## Setup / 形态
- **市场环境识别**：震荡行情常发生在一段大幅波动之后，价格急剧变化到新价位导致供需极度不平衡（00:00:41-00:52）
- **均线工具 — Vegas隧道**：
  - **EMA144** 指数移动平均线
  - **EMA169** 指数移动平均线
  - 两条线构成"震荡中轴"（00:01:47-01:54）
- **辅助指标**：MACD（用于查看背离）（00:03:05-03:09）
- **周期搭配**：15分钟交易级别 + 4小时趋势级别（00:02:10-02:15）

## Entry Rules / 入场
策略分5步：

**第1步 — 识别震荡区间**（00:01:34-02:05）：
- 价格围绕Vegas隧道EMA144/169上下波动
- 短期大幅波动后偏离Vegas隧道，趋于平稳后大概率形成"以隧道为中轴的对称性波动"
- 上方供给区阻力 + 下方需求区支撑构成通道

**第2步 — 判断大级别趋势**（00:02:05-02:30）：
- 4小时图表是下跌趋势 → 只在震荡区间**上方阻力位置入场做空**
- 4小时图表是上升趋势 → 只在下方支撑位置入场做多
- 只顺应大趋势做单方向

**第3步 — 多信号共振**（00:02:41-03:09）：
- 价格是否接近前高/前低/支撑/阻力
- K线形态结构出现（如双顶双底）
- 价格偏离均线的幅度过大
- MACD出现背离

**第4步 — 网格法分批入场**（00:03:10-04:05 示例）：
- 多头/空头将仓位分成几份，价格接近信号位置时分批入场
- 例：BTCUSDT 15分钟空单：
  - 矩形下方入 30%
  - 矩形上方入 30%
  - 前期波段顶部入 40%
- "前仓位等量、后仓位加量"，平摊风险、扩大止损范围

**第5步 — 突破操作**（00:04:16-04:44）：
- 震荡末期价格越来越收敛，可能选择方向或频繁扎针
- 突破震荡上沿出现大阳线 → 行情可能延续 → 及时转换思路顺势做单

## Exit & Stop Loss / 出场止损
（00:04:05-04:16）
- **止损**：设置在前期波段顶部的上方（空单）或对应低点下方（多单）
- **止盈**：
  - 选项A：在均线对称幅度的位置（即对面边界）
  - 选项B：幅度过大时，在Vegas隧道附近就开始分批止盈

## Risk Management / 仓位
- 网格分仓 30%/30%/40% 加重在更远的关键位置
- 不重仓单一价位

## Examples / 实例
- **BTCUSDT 15分钟**（00:03:23-04:15）：
  - 4小时大趋势下行
  - 价格短期大跌后上穿Vegas隧道，形成第一个波峰
  - 与波峰构成双顶
  - 第二个顶部价格回来 → 分批入场空单（30/30/40）
  - 止损：前期波段顶部上方
  - 止盈：均线对称幅度的位置

## Caveats / 注意
- 策略前提：短期大幅波动后大概率出现以Vegas隧道为中轴的、有明显边界的震荡区间
- 对无需震荡或难判定的走势"尽量不去参与"
- 大资金集团很爱在震荡末期清洗对手盘（13万人单日爆仓案例）— 突破方向不可逆抗
- "只做自己看得懂的行情也是一种智慧"（00:04:59）
