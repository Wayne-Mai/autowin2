# 周期+流动性+技术分析终极交易系统 — 半木夏

## TL;DR
半木夏引退前留下的终极策略，结合三个层面：周期（用MA30/120/256判断牛熊8阶段）、流动性（监控美联储资产负债表+TGA账户+重大事件+链上数据）、技术分析（结构形态指标）。核心：顺大势而为是首位，大势就是周期。

## Setup / 形态
- 资产：加密货币（比特币为主）
- 时间框架：日线为主，4小时辅助
- 工具：3条移动平均线（MA30、MA120、MA256）、MACD（13/34）、Vegas通道、斐波那契、固定成交量分布

## Entry Rules / 入场
**层面一：判断市场周期** (01:03)
- 8阶段周期：熊市→熊末→熊反→牛初→牛市→牛末→牛调→熊初
- 添加3条MA：MA30、MA120、MA256 (01:29)
- **走熊判断**：价格长时间处在自上而下空头排列的三均线下方
- **熊末判断**：均线开始拐头向上，价格反弹突破MA256，迎来熊市反弹
- **牛熊分界**：MA256均线附近，如果熊反的回调无法将价格压在均线下方 → 牛初来了
- **牛市判断**：价格长时间处在自下而上多头排列的三均线上方
- **牛末警示**：市场情绪指标极端、媒体社区言论异常一致、多空人数比异常、波动幅度和爆仓量越来越大；行情斜率明显增加快速冲高预期位置（如十万这种标志性位置）一定不能上头

**层面二：监控流动性** (03:20)
- 三大长期方向：
  1. **美联储资产负债表规模** — 大幅增加=美元大规模流入；大幅减少=美元抽离 (03:41)
  2. **美国财政部TGA账户余额** — 余额下降=注入资金；余额增加=财政在省钱或抛售 (04:15)
  3. **重大事件** — 以太坊ETF、比特币减半、美联储降息等 (04:42)
- 监测工具：
  - 交易所（观察24小时交易量、持仓量、订单价差和深度）(05:04)
  - 区块链浏览器（OKX浏览器、Glassnode、CryptoQuant等监测大户/灰度/富达/贝莱德地址）(05:26)
  - 监管机构调查报告 (05:59)

**层面三：技术分析** (06:21)
- **结构** = 密集成交区+重要高低点+斐波那契回撤+均线区域
  - 密集成交区：用固定成交量分布图寻找POC线，价格第一次触达时经常形成停止
  - 重要高低点：长上影线放量的前高、大量成交量堆积的波段低点
  - 均线：日线MA30正常回撤；MA120中线级别回撤；MA256震荡=牛市危险
  - 斐波那契：找一段重要趋势的起点与阶段高点相连，0.382和0.5附近是回调可能到达的位置
- **形态** = 收敛结构 + 头肩形态
  - 牛市回调第一波很猛，跌破第一波低点也不会差太多
  - 收敛形态是中继阶段，继续上涨概率高
  - 头肩入场在突破或回踩颈线的位置（普通和复合头肩都适用）
- **指标** = 半神背离（MACD快线13，慢线34）
  - 小时或4小时级别MACD与价格连续背离且**相差悬殊**
  - 在第二波峰或第三波峰的**实柱转虚柱所对应的K线收盘价入场** (08:43)

## Exit & Stop Loss / 出场止损
- **止损要果断**，果断才能占据主动 (08:56)
- 计算止损后加回来的差价，如Vegas通道：跌破止损、涨破开回，试错成本不高 (09:11)
- **止盈** (09:20)：
  - 斐波那契扩展
  - 前期重要关键区
  - **规定：跌破比特币4小时级别MA120超过2%就获利卖出或降低杠杆**
- 牛市可以卖出，尽量不去做空 (09:32)
- 危险或乱插乱震时：开个套保去吃费率套利
- 牛市资金费率年化收益甚至超过100%
