# 复利仓位管理表格 — Speculation Lab

## TL;DR
不是一套入场系统，而是一套仓位管理工具。每笔交易冒固定百分比风险（如 1%、2%、3%、10%），按固定风险回报比（如 1:2）博取盈利，把盈亏滚入下一笔订单本金。表格自动算止损金额和止盈金额，并生成两张可视化表格（风险回报表、账户资金增长表）。强调"一致性"是复利成败的关键。

## Setup / 形态
- 适用：任何品种、任何周期、任何策略
- 工具：Google Sheets 表格（视频说明栏提供）
- 表格结构：
  - 第 1 列：交易次数序号
  - 第 2 列：账户的本金
  - 第 3 列：风险比率
  - 第 4 列：每笔订单的止损金额
  - 第 5 列：每笔订单的止盈金额
- 右上方设置：风险比率数值（如 1%/2%/3%/10%）+ 风险回报比例（如 1:2）

## Entry Rules / 入场
**复利计算示例（01:53–02:50）**
- 账户本金 1000 美金
- 每笔交易冒 10% 风险 → 止损金额 100 美金
- 1:2 风险回报比 → 止盈目标 200 美金
- 若止盈：新本金 = 1000 + 200 = 1200
  - 下一笔止损金额 = 1200 × 10% = 120
  - 止盈金额 = 240
- 若亏损：新本金 = 1200 - 120 = 1080
  - 下一笔止损金额 = 1080 × 10% = 108
  - 止盈金额 = 216

**每次只需输入"账户本金"，表格自动计算所有数据**

## Exit & Stop Loss / 出场止损
- 入场时同时按风险金额定止损位
- 按风险回报比 1:2 定止盈位
- 表格不直接提供入场信号；与任何交易系统结合使用

## Risk Management / 仓位
- 固定风险百分比是核心：1%、2%、3%、10% 等任选
- 视频建议根据"对风险的厌恶程度"调整
- "一致性是复利成败的关键" — 即使胜率 60% 但 100% 执行的策略，比胜率 80% 但只执行 10% 的策略更有价值
- 好的资金管理可以弥补策略不足

## Examples / 实例
- 半神半木夏第二次从 1 万赚到百万仅用几个月，践行的就是复利效应
- 肥宅比特币胜率很低、频繁止损试错，但每次大行情来时都在场内 → 复利积累
- "肥宅说真正波动的时间只有 1%，其余 99% 在震荡。想抓住 1% 波动最好的策略是熬过那 99% 的震荡时间"

## Caveats / 注意
- 表格只是辅助工具，应用好坏看交易员自身
- 不要像"日记本一样搁置在一边"
- 一致性比策略本身重要
- 即使有完美策略，没有纪律也免不了陷入困境
- 必须先在图表大量复盘 → 小资金试水 → 确认能盈利后再正常交易
- 心理因素（冲动交易）会让任何好策略失效
- "你搁着它，它就不属于你"
