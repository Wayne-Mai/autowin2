# 突破追高强势股短线策略 — 徐翔

## TL;DR
徐翔早期（散户期）的核心策略：在涨跌停板制度下找出有题材+新闻+量价齐升+封单难度大的龙头股，用突破追高强势股的方式做短线。22岁用3万滚到300万。3个月内画了3000张图纸才摸索出来。

## Setup / 形态
- 资产：A股
- 背景：1996年12月沪深两市开始实施涨跌停板制度（单日涨跌不超过10%）
- 逻辑：原本计划涨50%的股票，10%涨停后还有40%空间分几天涨完 → 庄家的马脚 (02:21)

## Entry Rules / 入场
**徐翔早期策略规则**（3个月画3000张图纸摸索出来）(02:33)
1. **密切关注题材新颖的龙头股** (02:40)
   - 最好有消息面支撑
   - 题材新颖的龙头 + 引起广泛关注的新闻 = 吸引更多关注和资金流入
2. **封单难度大** (02:54)
   - 出现连续涨停
   - 量价齐升突破一些关键位置
3. **量价齐升突破关键位置 = 大资金实实在在入场推动价格的证明** (02:58)
   - 继续上涨的概率很大

## Exit & Stop Loss / 出场止损
- 视频未详述徐翔的具体止盈止损规则
- 由于其调仓习惯激进凌厉（大多数时间在全仓与空仓之间对调），可推测其出场是全部清仓 (01:25)

## Risk Management / 仓位
- 大多数时候全仓或空仓对调
- 这种调仓习惯贯穿整个交易生涯（即使资金庞大到几十上百亿时也是如此）

## Examples / 实例
- 1999年五一九暴涨行情 → 账户做到300多万
- 这个时期被人称为"股市敢死侯"

## Caveats / 注意
- 这只是徐翔三部曲的第一部，2000年后市场低迷、突破追高强势股策略经常被套，他随之进化到联合大户封涨停板的2.0玩法（散户跟随会被套）
- 2.0玩法：选市值小、热度高的股票，短时间集结大资金人为制造妖股，大单高价买入直接封涨停板，第二天可能继续封形成连板，吸引散户后高位出货
- 3.0玩法（泽熙时期）：联合大股东组局，通过股东身份与大股东商谈方案出利好（最常用抛出高送转方案的饼），等股价上涨后股东高位套现 — 该层级有违规风险，徐翔2015年被捕，2021年7月9日出狱，刑期5年6月，被罚没200亿
- 缺乏稳定性的策略坚持不了多久 — 徐翔本人在2000年市场低迷时也意识到需要换策略
