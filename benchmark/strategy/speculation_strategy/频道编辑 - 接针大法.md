# 接针大法 — 频道编辑

## TL;DR
币圈极度缺乏监管+信息不对称，大资金机构经常打掉散户止损/爆仓订单后价格立刻回归，留下长长引线（针）。接针就是跟随大资金团伙做短线，接到一根针可获得1xx%瞬时收益。三种适合接针的位置：趋势盘整中的扎针、Vegas等均线回归、重要支撑阻力位。风险极高，可能直接爆仓。

## Setup / 形态
- 资产：加密货币（合约市场）
- 时间框架：日线/30分钟/盘中
- 工具：Vegas隧道（EMA144、EMA169）、MA90、EMA200、多空比、持仓量

## Entry Rules / 入场
**接针三种位置**

**位置一：趋势运行的盘整过程中扎针** (01:31)
- 多单例子：价格上涨后开始震荡盘整
  - 同方向资金太多（如盘整区间内多空比从1涨到2）= 散户开单条件齐全
  - 散户止损单集中放在震荡区间下沿
  - 大资金扎针快速吃掉这部分筹码后继续推进
  - 价格打下来时持仓量出现断崖式下跌 → 确认
- 空单例子：下跌走势的盘整过程，多空比持续增加+急涨大阳线 = 可能是扎针
  - 针打穿不只是当前震荡区间上沿，而是直接打到上一个结构位置（特征明显）

**位置二：均线回归** (02:43)
- 用MA90、EMA200或Vegas隧道（EMA144/169）
- 当趋势行情中价格逐渐偏离均线，回归均线时已消耗大量订单
- 多头例子：价格突破EMA144/169 Vegas隧道后拉升远离 → 忽然急跌打到隧道附近留下长下引线 → 这针约3%利润，10倍杠杆=30%收益
- 空头例子：价格跌破Vegas隧道后下跌远离 → 反弹第一次接近Vegas隧道时接针

**位置三：重要的支撑和阻力位置** (04:57)
- 阻力支撑级别越大、价格偏离越远的急涨急跌大K线第一次打到这里，成针概率越大
- 比特币日线案例：2021-05-19大阴线从43600直接打到30000整数关口（同时是前期结构支撑），跌幅30%以上，场内几乎见不到幸存的多头 → 巨大急跌大阴线 + 偏离很远的重要大级别结构位 = 接针好位置
- 切到30分钟可见针扎的非常标准
- 2020-03-13触底大针：欧阳接到这个位置，明牌持仓很长时间，最终因交易所问题强平，赚近100万美金

## Exit & Stop Loss / 出场止损
- **接针单子达到一定利润后一定不要贪心**，快进快出 (03:42)
- 瞬时3%利润已经非常可观（加杠杆10倍=30%）
- **开单后必须立刻设置止损**
- Vegas隧道EMA144/169之间幅度小，想接幅度可观的行情需要把止损放宽
  - 例：止损放在隧道下方1%-2%位置

## Risk Management / 仓位
- **仓位一定要小**
- 接针前必须有止损计划
- 散户杠杆基本5倍以上，但接针时建议低杠杆
- 担心快进快出后悔的可以准备两个账户：现货账户做长线 + 合约账户做短线，互不干扰 (06:23)

## Examples / 实例
- BTC日线 2021-05-19：43600 → 30000，30%+跌幅，30分钟级别针扎标准
- BTC 2020-03-13触底大针：欧阳赚近100万美金
- Vegas通道接针：典型针3%瞬时利润

## Caveats / 注意
- 接针是非常暴力的跟随大资金交易方式，风险系数很高
- 极端行情可能直接爆仓
- "绝户针"风险：交易所可能直接打穿Vegas通道甚至到前低，超过20%的针让5倍杠杆以上用户无人幸免
- 散户操作习惯一致 → 容易被大资金针对
- 散户的杠杆基本会开到5倍以上，需要警惕
