# SpeculationONE 指标套件 — 投机实验室

## TL;DR
投机实验室出品的多功能 TradingView 指标，集成了半神 MACD、Vegas 通道、FVG、订单块、主力大单成交、CME 缺口监测和共振表。可用 MACD 双/三背离 + Vegas + 缺口 + 订单块 + 大单成交聚集作为共振入场信号。

## What It Measures / 它衡量什么
1. **CME 缺口监测** — 比特币 CME 期货周末跳空形成的缺口，历史回补率接近 100%
2. **主力大单成交** — 价格区间内快速聚集的成交大单，红/黄/蓝三色圆圈表示大/中/小（仅 BTC，1H 胜率最高）
3. **半神 MACD (13, 34)** — MACD 柱状图背离寻找信号，能量柱绿/红表示看涨/看跌
4. **Vegas 通道** — 5 条 EMA (12, 144, 169, 576, 676)；EMA12 默认隐藏
5. **FVG 公允价值区** — K 线急速突进留下的价格不平衡缺口
6. **订单块** — 大资金集中进场后形成的供需区
7. **共振表** — 各指标信号同向时亮灯

## Parameters & Settings / 参数设置
- 半神 MACD 默认：快线 13，慢线 34
- Vegas EMA：12 / 144 / 169 / 576 / 676（EMA12 默认隐藏）
- 主力大单：默认中等单量组（最均衡）；调到最大 → 信号少但关键节点胜率更高
- 调节波峰大小差异、最小高度等参数适配不同品种和时间周期

## How To Read / 怎么解读

**CME 缺口（00:24–01:51）**
- BTC 行情中 CME 缺口回补率接近 100%
- 提前在缺口位置布订单
- 视频例：4H BTC 形成宽 CME 缺口 → 缺口形成后又拉升约 40% → 4 个月内回落补齐缺口；回补同时 MACD 形成连续三背离 → 1 天 15% 利润，止盈点在上方缺口聚集区

**主力大单（01:53–03:03）**
- BTC 1H 级别胜率最高
- 红色 = 最大成交大单（关键节点）
- 黄色 = 中等 / 蓝色 = 最小
- 关键信号：**两个或三个包含红圈的密集区域**
- 越红越密效果越好 → 提前挂订单吃瞬时利润

**半神 MACD 背离（03:17–04:56）**
- 三背离规则：零线下方有 3 个柱状波谷相差较大且**连续升高** + MACD 对应的 K 线波谷**连续降低**（多头）
- 条件满足，MACD 实心柱转虚心柱的**第一根柱子收盘后**出现买入信号
- 这根虚柱对应的 K 线 = 关键 K 线
- 在关键 K 线**下一根 K 线的开盘价附近**入场多单
- 三个决定成败的细节：
  1. 只找波峰相差较大的连续背离，不连续不要
  2. 波峰越标准越好，奇形怪状不要
  3. 三背离做中长线，双背离只做短线

**双背离 vs 三背离**：
- 三背离：胜率高、力度大、频率低，出现后非常值得参与（中长线）
- 双背离：频率高得多但力度较小，只做大市值品种短线
- BTC 日线双背离：从最低点 1 万多到现在 12 万共 12 次机会，9 次成功止盈，胜率超 75%

**Vegas 通道（07:01–07:40）**
- 多头：EMA144/169 在 EMA576/676 上方 + K 线触碰 EMA144/169 通道 → K 线上方显示绿色三角，共振表 Vegas 一栏亮绿灯
- 空头：相反

**FVG（07:40–09:12）**
- 三种 FVG 特别重要：
  1. **结构突破后出现的 FVG**：胜率远高于其他形态（如上升趋势突破前高后附近的 FVG，价格大概率回撤到缺口再反弹）
  2. **同区域多个 FVG 时选最大的那个**：越大多空失衡越明显，把中线作入场参考
  3. **避开靠近重要关键区的 FVG**：小周期看涨缺口但切到大周期上方有重要阻力 → 这种不碰

**订单块（09:13–10:09）**
- 看涨订单块通常在上升趋势中的局部或底部
- 看跌订单块通常在下降趋势中的局部或顶部
- 配合共振信号确认：
  - Vegas 通道 144/169 均线作为关键区共振
  - MACD 背离
  - 大单成交聚集

## Trade Setups Using It / 配套的入场出场
**MACD 背离标准入场（多）**:
- 入场：关键 K 线下一根 K 线的开盘价附近
- 止损：波段低点附近 或 关键支撑位下方
- 止盈幅度：止损幅度的 1.2 - 1.5 倍
- 到第一目标平掉一半仓位，止损上移；剩余拿 1:2 甚至更高

**MACD 背离标准入场（空）**：相反

**共振信号**：
- Vegas + MACD 都出多头 → 两灯同时亮，胜率更高
- MACD 背离 + 大单成交聚集 + 订单块 → 高质量共振

## Caveats / 注意
- 不连续的背离不要
- 奇形怪状的波峰不要
- 双背离只做大市值短线
- 主力大单目前只开发了 BTC（ETH 还在调试）
- 必须自己回测不同品种/周期找最适合的组合
- 节目仅供娱乐，无投资建议
- 指标获取需要填写谷歌表单申请授权（免费/付费两种）
