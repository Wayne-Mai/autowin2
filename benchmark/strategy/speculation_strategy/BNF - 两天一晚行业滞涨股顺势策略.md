# 两天一晚行业滞涨股顺势策略 — BNF（小手川隆）

## TL;DR
2003年日本股市转牛后，BNF从逆向投资切换到顺势策略：发现行业内一家主要公司开始与日经指数同步上涨时，立刻买入同行业内"还没上涨"的滞涨股。每天持30-50支股票，过夜后第二天早上获利/止损 → 资产从1亿日元飙至80亿日元。

## Setup / 形态
- **市场环境**：牛市 / 2003小泉改革+全球经济复苏后日股转牛
- **品种**：日股
- **持仓周期**："两天一晚"（00:06:22-06:32）
  - 每天持有 **30-50支股票**
  - 过夜后第二天早上盈利确认或止损
  - 寻找下一个强势股票持有
- **核心理念**：行情好时顺势而为；BNF坦言自己没有日内交易的天赋（00:06:14-06:20）

## Entry Rules / 入场
**核心方法 — 行业联动 + 滞涨股**（00:06:36-07:02）：

1. **监控行业内主要公司**：
   - 在钢铁行业中（举例），关注四大主要公司
   - 一旦发现其中一家**与日经平均指数一起开始上涨**，这就是入场信号

2. **买入同行业还没涨的滞涨股**（深度解析版 00:03:33-03:51）：
   - 行业联动：一家涨 → 同行业其他还没涨的滞涨股大概率会跟涨
   - 例：钢铁四大公司其中一家先动，将目光投向另外三家
   - 买入符合条件的滞涨股，搭上整个行业的上涨波段

3. **同时持有多支股票**：
   - 每天 **20-50支**（不同视频版本数值略有差异，但都是大数量）
   - 分散单股波动风险，达成总体盈利目标

## Exit & Stop Loss / 出场止损
- **第二天早上**：选择获利了结或止损（00:06:25-06:32）
- 严格执行循环操作
- 视频未明示具体止损百分比

## Risk Management / 仓位
- 通过同时持有 **20-50支股票** 来分散风险（不集中在单一标的）
- 一晚持仓 → 第二天处理
- 频繁的交易会带给自己过多喜悦和担忧 → 所以不做日内 → 选用两天一晚

## Examples / 实例
- 2002年底 **1亿日元 → 2005年底 80亿日元**
- 2005年 **J-com 误下单事件**：10分钟内赚 **20亿日元**
- 后续购入东京秋叶原、涩谷等黄金地段多栋大楼

## Caveats / 注意
- **生活极简**：没出国旅行过、自行车交通、常吃泡面、超市买衣服、百元店买日用品（00:07:22-07:33）
- "对增加财富感兴趣，但对花钱完全没有兴趣"
- "当我看到面前堆积的现金时，就会变得无法在关键时刻做出冷静的判断，因此我尽量避免看到现金"
- **关键认知**："当市场情况不佳时，逆向投资策略非常有利；当市场行情良好时，应该顺势而为"（00:06:00-06:10）
- 要把握的行业内股票信息量巨大
- 需要切换到顺势 = 牛市信号判读+行业联动监控
