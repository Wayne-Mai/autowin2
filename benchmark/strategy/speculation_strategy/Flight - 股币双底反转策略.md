# 股币双底反转策略 — Flight

## TL;DR
Flight 指出股票和加密货币的双底结构不同：股票通常**左底低右底高**走势规律；加密货币因庄家"打完前低附近的止损订单后才上行的概率非常高"→ 双底常呈**右底低于左底**（破前低假摔）。该策略要求结合 RSI 或 MACD 背离 + Vegas 通道作为共振确认。

## Setup / 形态
**股票 vs 加密货币双底差异**（00:04:42-04:55）：
- **股票双底**：左底低、右底高（相对规律）
- **加密货币双底**：庄家道德低下，打完前低附近的止损订单后才开始上行的概率非常高
- 也就是说币圈的"右底破前低"是常态，是庄家洗盘后真正反转的信号
- 这种双底形态最好**结合其他策略一起使用**

## Entry Rules / 入场

**步骤1 — 形态识别**（00:05:07-05:21）：
1. 价格急剧下跌
2. 在关键区域附近出现**巨量K线**形成**左底**
3. 之后又跌破前低 → 形成"标准的加密货币双底结构"
4. 在双底结构的**颈线位置上方处**入场

**步骤2 — 信号共振（任选）**：

**A. 配合 RSI**（00:05:21-05:31）：
- RSI 出现背离后
- 离开超卖区所对应的 K 线收盘价入场多单

**B. 配合 MACD**（00:05:31-05:53）：
- MACD 快线 = **13**
- MACD 慢线 = **34**
- 当价格在关键区域附近形成双底（比如 **Vegas 通道附近**）
- 在 MACD 连续背离且相差悬殊的**第二波峰的实柱转虚柱**所对应的 K 线收盘价位置入场多单

## Exit & Stop Loss / 出场止损
（00:05:53-05:59）
- **止损**：
  - 设置在**近期低点附近**
  - 或 **Vegas 通道下方**
- 止盈：视频未明确，沿用 Flight 的高倍率播种框架（分批止盈或到达预期价格平仓）

## Risk Management / 仓位
- 沿用 Flight 高倍率播种框架：
  - 1% 本金 × 20 倍杠杆
  - 单笔止损 2%
  - 单日最大开 10 次
  - 单笔同时开仓最多 3 个标的

## Caveats / 注意
- **不能直接套用股票双底规则**到加密货币
- 加密货币庄家会"先破前低洗止损"再拉升 → 右底比左底低是常态
- 必须与其他信号共振：
  - RSI 背离离开超卖
  - 或 MACD(13,34) 第二波峰实柱转虚柱
- Vegas 通道附近双底效果最好
- 是 Flight 五大策略中的第5种
