# Smart Money Concepts [LUX] 聪明钱概念综合指标 — Speculation Lab

## TL;DR
SMC [LUX] 是一个自动标记市场结构（BOS/CHOCH）、订单块、公允价值差距 FVG、溢价折扣区的全能价格行为指标。核心理念是跟随机构/大资金交易，在上升趋势中只在折扣区买入，下跌行情中只在溢价区卖出。

## What It Measures / 它衡量什么
- 自动标记市场结构的全能综合指标
- 订单块高低点、公允区域、折扣溢价区
- 跟随聪明钱交易（机构/大资金集团交易者比散户更聪明）
- 找到机构在市场交易的痕迹

## Parameters & Settings / 参数设置

### 安装
- 指标搜索栏输入 "Smart Money Concepts [LUX]"
- 收藏并添加到图表

### 模式选项（01:16-01:25）
- **历史**（默认）或 **当前**
- 当前 = 只在图表显示最近的数据
- **风格**：默认彩色 或 单色

## How To Read / 怎么解读

### 内部结构（01:25-01:48）
**BOS（Break of Structure）**：
- 价格突破之前上升趋势的高点
- 或跌破之前下降趋势的低点
- 趋势延续信号

**CHOCH（Change of Character）**：
- 价格跌破上升趋势中较高的低点
- 或突破下降趋势中较低的高点
- **潜在的逆转迹象**

### 摆动市场结构（01:48-02:08）
- 显示更大级别的市场结构
- 标签相同但用**实线**表示（与虚线内部结构区分）
- 打开"显示摆动点" → 标记更高高点、更高低点、更低高点、更低低点
- 趋势一目了然

### 订单块（02:10-02:50）
- 类似供给需求区
- 机构、大资金集团容易参与的价格区域
- 既可作支撑阻力，也可寻找止盈/止损
- **BTC/USD 15 分钟示例**：
  - 价格突破红色订单块回踩后做多
  - 或等价格跌破红色订单块择机做空
  - 止损：近期高点附近或订单块上沿
  - 止盈：下方蓝色订单块附近

### 等高等低点（02:50-02:59）
- 显示类似双顶/双底的结构

### 公允价值差距 FVG（02:59-03:47）
- 标记图表上的不平衡区域
- 包括：实体大 K 线、跳空、连续上涨/下跌的 K 线
- 这些是流动性不平衡的范畴
- **验证方法**：打开固定范围成交量分布图选中 K 线 → 标记 FVG 部分内部成交量较小 → 说明该区域卖单/买单严重不足
- **意义**：流动性不平衡是大资金入场信号
- 大机构操盘算法大概率会引导价格立即或将来填补这些区域 → 让市场保持平衡
- 这些价格区域更容易出现盈利机会

### 高低点 MTF（03:47-04:11）
- 标记前一天 / 前一周 / 前一个月的高点和低点
- **日内交易者**：前一天最高最低价
- **1h/4h 中短线交易者**：前一周/前一个月最高最低价

### 溢价折扣区（04:11-04:42）
- 打折低价 → 有人买入
- 涨得过高 → 有人卖出
- 与散户追涨杀跌有本质区别
- **应用规则**：
  - 上升趋势中 → 只在打折区买入
  - 下跌行情中 → 只在溢价区卖出

## Trade Setups Using It / 配套的入场出场
- 入场：订单块附近 + BOS/CHOCH 信号
- 在 FVG 不平衡区附近寻找回填机会
- 止损：订单块上沿或近期高低点
- 止盈：下一个订单块

## Caveats / 注意
- 唯一不足：少一个枢纽点功能 → 需配合 Pivot Points Standard 使用（04:49-04:58）
- 必须结合价格行为分析，单靠指标标记不够
