# 加密市场三大数据指标 — Speculation Lab

## TL;DR
这是一套基于 OKX 交易所公开数据的市场情绪/资金行为工具，包含多空持仓人数比（散户情绪）、精英多空平均持仓比（前 100 名大户情绪）、持仓量与成交量。视频的核心是"散户做多多就反向走，大资金与散户背离时跟大资金"，并通过持仓量崩塌识别行情停滞节点。

## What It Measures / 它衡量什么
- **多空持仓人数比**：合约市场中做多人数 ÷ 做空人数。资金 1:1 等额，但人数差能反映散户情绪。
  - 大于 1 → 散户做多情绪偏强
  - 小于 1 → 散户做空情绪偏强
- **精英多空平均持仓比（OKX 前 100 名）**：反映大资金/大户的仓位方向（按持仓金额而非人数加权）。
- **持仓量（Open Interest）+ 成交量**：持仓量反映多空参与者在该时点的总持仓；成交量反映当时的换手活动。

## Parameters & Settings / 参数设置
- 视频中所有数据均直接使用 OKX 提供的默认数据流，没有自定义参数。
- 数据查看入口在 OKX 的数据/合约持仓数据页面。

## How To Read / 怎么解读
**多空持仓人数比与价格的配合（02:21–03:30）**
- 价格快速上涨 + 多空人数比也增长过快 → 散户在追涨，价格大概率在下一个阻力位遇阻回落。
- 价格快速下跌 + 多空人数比仍持续高位 → 抄底的散户太多，"大家都认为的底大概率不是底"，价格大概率继续下跌或宽幅震荡。视频以 BTC 在 3 万美金附近震荡一个多月为例，OKX 的多空比每次回落后又快速回到高点，说明散户越跌越买，行情不会启动。
- 与价格走势配合异常时容易出现极端行情。

**精英多空平均持仓比的背离策略（04:04–04:50）**
- 精英多头强 + 散户空头强 → 倾向跟随大资金做多，尤其在暴跌后处于行情分水岭、市场情绪悲观时。
- 散户多头强 + 精英空头强 → 市场可能反转。如果价格刚好在高位或关键阻力附近，要警惕剧烈波动，避免高杠杆或开套保对冲。

**持仓量 + 成交量（04:53–05:42）**
- 行情进行到关键区域附近时，如果"成交量很大但持仓量出现断崖式下跌"，说明一方大部分仓位已平或被强平，大资金机构扫清了流动性 → 行情大概率停滞，出现止跌反弹或遇阻回落。

## Trade Setups Using It / 配套的入场出场
视频没有给出明确的入场价/止损价规则，只给出方向性偏置：
- 散户追多 + 大资金做空在阻力位 → 偏空或减仓
- 散户追空 + 大资金做多在支撑位 → 偏多或抄底准备
- 持仓量崩塌 + 成交量天量 → 准备反向操作或离场观望

视频明确建议结合其他技术指标或价格行为分析作为入场触发。

## Caveats / 注意
- 单个数据展现的角度不全面，必须组合使用并结合技术分析（05:51–05:58）。
- 这是市场情绪/资金侧信息，不是直接的入场信号。
- 大资金不等于一定对，"大资金交易者可能会有更多信息和判断经验"，但视频也强调要"借助其他指标或工具去印证，不要急着入场"。
- 节目结尾明确"没有任何投资建议，纯属娱乐"。
