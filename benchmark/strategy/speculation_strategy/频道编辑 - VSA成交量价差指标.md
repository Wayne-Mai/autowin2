# Volume Spread Analysis (VSA) 成交量价差指标 — 频道编辑

## TL;DR
VSA (Volume Spread for VSA) 是一个**领先指标**——交易量的快速增加会影响价格原有走向的概率很大，可以在价格可能发生停止行为前发出预警信号。单指标 BTC 5分钟图 100 次连续交易胜率超过 66%。最神奇的玩法：**红色超高量能柱对应大阴线时入场多单 / 大阳线时入场空单**——左侧交易、猜顶摸底。

## What It Measures / 它衡量什么
- 测量每根K线的成交量与价差(spread)的关系。
- 超高量能 + 大K线 → 该价位附近被大量交易者下注（资金量大），后续价格大概率会被这些被套订单/原有订单产生的"停止行为"反向影响。
- 是**领先指标**——可在价格可能发生停止行为前发出预警 (00:51)。

## Parameters & Settings / 参数设置
- 指标名：**Volume Spread for VSA** (00:26) (后面统称 VSA)
- 设置 (00:34)：
  - 其他用不到的功能勾掉，只留 **量能柱主体**
  - **超高量能 = 红色**
  - 其他颜色 = **蓝色**
- 配套：**ATR Stop Loss Finder** (01:41)
  - **倍数 = 1.7**
  - 其他参数保持不变

## How To Read / 怎么解读

### 红色超高量能柱 = 警示信号 (00:57)
- 红色超高量能柱对应**大阴线** → 价格在低位被大量买单接盘 → 可能反弹 → 入场**多单** (01:00)
- 红色超高量能柱对应**大阳线** → 价格在高位被大量卖单压制 → 可能回落 → 入场**空单** (01:14)
- 多根连续红色超高量能柱 → 只做**第一笔**交易 (01:22)
- 视频作者反复强调："是在大阴线的收盘价入场多单，你没有听错" (01:06)——这是左侧交易，有猜顶摸底的嫌疑。

### 三种典型场景

**1. 顶部/底部反转** (02:38)
- 一段明显趋势中，价格连续两次冲高 → 在前高附近出现红色超高量能柱 → 卖盘订单在该位置聚集 → 大概率冲高回落 (03:08)
- 一段下跌中，价格连续两次触底 → 在前低附近出现超高红色量能柱 → 红色量能柱对应K线收盘价做多 (03:18)

**2. 双顶/双底结构 + VSA共振** (02:43)
- 这种交易机会不多，但出现后胜率很高 (02:59)

**3. 密集成交区** (03:31)
- 三根超高量能柱一起出现 → 价格范围吸引了大量交易资金多空换手 → 当价格再次接近该密集成交区时，尤其第一次会遇阻回落 (03:35)
- 这种"短线策略"必须果断止盈，不能贪心 (04:10)

## Trade Setups Using It / 配套的入场出场

### 标准入场（左侧交易）
- 红色超高量能柱对应大阴线 → 大阴线收盘价做多
- 红色超高量能柱对应大阳线 → 大阳线收盘价做空
- 连续红色超高量能柱 → 只做第一笔

### 止损 (01:33)
- 波段高/低点；或
- **ATR Stop Loss Finder（倍数1.7）**
  - 多单：ATR 下线位置止损
  - 空单：ATR 上线位置止损

### 止盈
- 分批止盈方式 (02:04)：1:1 出一部分，剩余留在场内博取更大利润。

### 单边过滤
- 大趋势是多：只在小级别图表找波段下沿做多 (02:28)
- 大趋势是空：只在小级别图表找波段上沿做空 (02:32)
- VSA 比较适合震荡行情的波段交易；趋势行情容易被连续止损 (02:19)。

## Caveats / 注意
- VSA 属于**左侧交易**，猜顶摸底——心理压力大、容易扛单亏损 (02:16)。
- 趋势行情容易被连续止损，必须只做单边 (02:24)。
- 顶/底反转和密集成交区场景出现频率较低，但胜率高。
- 短线策略——目标达成必须果断止盈，不能贪心 (04:13)。
