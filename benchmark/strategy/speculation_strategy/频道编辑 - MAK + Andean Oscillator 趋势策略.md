# MAK + Andean Oscillator 趋势策略 — 频道编辑

## TL;DR
MAK (Multi Oscillator Adaptive Kernel) 通过自适应核算法把 Stochastic、RSI、MFI、CCI 四种指标融合，用于多重过滤震荡噪音。配合 Andean Oscillator 三条线（绿/红/橙）的穿越作为入场信号，在 BTC 4H 图表表现优秀，趋势行情中利润丰厚 (00:00-02:30)。

## Setup / 形态
- 指标 1：Multi Oscillator Adaptive Kernel（MAK），添加图表后将四个子指标参数全部设为 **200**（不敏感、大波动时不频繁改变信号）(00:46-01:03)
- MAK 中心线上方蓝色 = 上升趋势；中心线下方红色 = 下降趋势 (01:03-01:13)
- 指标 2：Andean Oscillator（AO），参数保持默认。衡量上升/下降趋势的波动程度，突出趋势方向和强度 (01:13-01:26)
- 时间周期：BTC 4H 图表 (01:33)

## Entry Rules / 入场

### 多头规则
1. MAK 处在中心线上方蓝色区域 (01:37)
2. 最好选择已有小段上升并平稳上涨的行情 (01:41)
3. AO 指标的绿线需先穿过红线，再穿过橙线 (01:46-01:51)
4. 此时出现买入信号，对应 K 线 = 关键 K 线，收盘价入场 (01:51-01:58)

### 空头规则
1. MAK 处在中心线下方红色区域 (02:01-02:08)
2. 已有小段上升后转为平稳下降走势 (02:08-02:14)
3. AO 红线先穿过绿线再穿过橙线 (02:14-02:20)
4. 关键 K 线收盘价入场空单 (02:21-02:25)
