# 高动能跟踪趋势模型 — Fabio Valentini

## TL;DR
Fabio 的两个高频模型之一，专用于**趋势市场**。利用 VP 成交量分布图找到突破波段内的**低成交量节点（LVN）**作为回撤入场位，并用订单流验证动能。止盈固定到下一个 POC 位置（成交量最密集的红线，反转/盘整概率超过70%）。

## Setup / 形态
- **适用时段**：波动最大的**纽约交易时段**（北京时间晚上8-9点开始）（00:00:49-00:54）
- **核心工具**：
  - **VPVR / VP 成交量分布指标**（TradingView）
  - 订单流 / 订单簿
  - CVD 成交量增量
  - 可选：SpeculationONE 工具的大单成交圆圈

## Entry Rules / 入场

**步骤1 — 识别市场状态**（00:00:56-01:09）：
- 大前提：**价格明确突破当前盘整区间**
- 突破后才能谈是趋势行情
- 避免在无方向震荡中被反复止损

**步骤2 — 找关键位置（低成交量节点 LVN）**（00:01:09-01:42）：
- 找到制造突破的"发起点结构"（趋势发起的源头）
- 用 VP 指标从左往右连接**源头低点**到**突破高点**
- 图表显示该波段详细的成交量节点
- **标记低成交量节点（LVN）**
- 价格突破时这些位置阻力很低，基本不会停留 → 后期经常回补
- 这是非常**理想的回撤入场位**
- **不要提前挂限价单**，尽量盯盘选择入场机会

**步骤3 — 确认动能是否主动**（00:01:42-02:16）：
- 当价格回撤到低成交量节点时观察订单流（或订单簿）
- 必须看到**与趋势方向一致的大额市价订单快速成交**
  - 市价单代表**主动**和侵略性
- 做空 → 找明显的主动大卖单成交订单
- 做多 → 找主动大买单成交
- 有 SpeculationONE 工具的可观察是否有圆圈出现
- 没有这种动能 = **不能入场**

**辅助 — CVD 验证趋势方向**（00:02:20-02:57）：
- CVD 是市场压力探测器
- CVD 下跌 + 价格上涨 = 短期顶背离（卖压累积）
- CVD 上涨 + 价格下跌 = 短期底背离（买盘累积）
- 成交量领先价格 → CVD 可提前发出反转信号
- 这种时候将止损移到保本位置或提前离场

## Exit & Stop Loss / 出场止损
**止损**（00:02:16-02:20）：
- 设置在**主动成交订单之后**
- 最好留一些缓冲空间

**止盈**（00:03:00-03:22）：
- **设置在前一个波段成交量分布图的 POC 位置**
- POC = 成交量最密集的红线
- 价格到达 POC 时**全部平仓**
- Fabio 说他观察到 **超过70%的概率会在POC发生反转或进入新的盘整**

## Caveats / 注意
- 必须先确认市场处于**趋势状态**才能用此模型
- 没有动能信号绝对不入场
- 不预挂限价单 — 必须盯盘确认
- 该模型适合纽约时段，伦敦时段或波动小时段不适用 → 切换到均值回归模型（见配套文件）
- POC反转概率仅为70%概率参考，并非保证
