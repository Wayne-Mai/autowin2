# Session Volume Profile (SVP) 成交量分布策略 — Speculation Lab

## TL;DR
SVP 是每日成交量分布指标，通过观察 POC（控制点）的连续走低/抬高来识别真正的密集需求区/供给区。POC 走势改变是趋势反转的客观信号，比主观划线更精确。

## What It Measures / 它衡量什么
- 每一天的成交量分布情况
- POC（Point of Control）= 当日成交量最高的价格水平
- POC 红线分布反映市场对哪些价格位置感兴趣的程度
- "量告诉我们的信息甚至比 K 线都要真实的多"（02:10）

## Parameters & Settings / 参数设置
- 交易级别：**15 分钟以上**（00:42）
- TradingView 布局技巧（02:14-02:25）：右上方布局按钮 → 左右两个布局；左边隐藏 K 线的成交量分布图；右边普通 K 线图

## How To Read / 怎么解读

### 真实密集需求区识别（00:48-01:31）
1. 比特币下跌波段中，POC 一直在下降 → 每天都有很多人认为价格还会跌得更低
2. 直到 POC 开始回升 → 没有再出现更低的 POC 线
3. 表示更多资金认为当前价格已经足够便宜，买已经比卖合适
4. 控制点回到多头手里 → 这个真实密集成交区就是密集需求区间
5. **经验规则**：POC 连续降低**一周以上**，下跌幅度越大 → 密集需求区出现后的支撑效果就越强
6. 价格再次到达这个需求区附近时，产生停止行为的概率就越大

### 真实密集供给区识别（01:31-01:53）
- POC 连续抬升**一周以上**
- 第一次出现回落时
- 这根回落的 POC 线与之前集中的 POC 形成的区间 → 密集供给区
- 价格再次回到这里遇阻回落的概率非常大

### 趋势判断（02:52-03:08）
- 价格突破后 POC 一直在抬高 → 交易者购买意愿增加
- 等 POC 幅度增加不再流畅 → 多头趋势可能要面临调整

## Trade Setups Using It / 配套的入场出场
- 在 POC 聚集形成的密集供需区附近寻找入场点
- 真实密集成交区附近汇集大量主动买卖订单 + 被动止损订单
- POC 越密集力量越大
- 配合 K 线形态做停止行为确认

## Caveats / 注意
- 主观划分的支撑阻力没有数据支撑，SVP 划分的真实密集成交区更精确
- 必须连续观察"一周以上"才能确认 POC 趋势
