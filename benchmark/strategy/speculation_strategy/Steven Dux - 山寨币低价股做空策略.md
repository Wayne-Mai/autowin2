# 山寨币/低价股做空日内策略 — Steven Dux

## TL;DR
Steven Dux专门做空低价仙股（市值小/波动高/流动性差/监管不严），也适用加密山寨币。底层逻辑：低价股基本面差，资金不会长期持有，拉盘就是为了出货，天然具有空头属性。用TradingView筛选器筛选符合条件的标的，等价格暴涨到关键阻力区+巨大成交量+多头攻势耗尽后入场做空。

## Setup / 形态
- 资产：美股仙股（"华尔街之狼"里的低价股）+ 加密货币山寨币
- 仙股定义：价格低、市值小、风险高、波动性大、流动性差、监管不严
- 时间框架：日内、1-15分钟、1小时
- 工具：TradingView筛选器

## Entry Rules / 入场
**TradingView筛选器设置**

**第一步：粗筛市值/成交量异常涨幅** (01:36, 05:43)
- 市值范围：**1亿到10亿美金之间**（≤3亿市值最好）
- 24小时成交量：**1000万美金以上**
- 从开盘涨跌百分比条件：**4小时5%或10%以上**
- 24小时涨跌幅度降序，让涨幅突出币排前面
- 例：市值1亿美元左右、24小时成交量1000万以上的标的
- 加上一小时或15分钟涨幅就超过5%的过滤

**第二步：搜消息验证非基本面** (06:08)
- 在coinmarketcap/财经媒体平台搜这些币种是否因炒作或非基本面因素被推高
- 有条件的可在大户社群看小道消息
- 大资金常用新闻和虚假热点人为推高价格 → 喊人接盘 → 迅速抛售
- 如果没有消息或利好消息异常 → 进入第三步

**第三步：标记关键区域 + 长上影线** (06:29)
- 标记币种近期关键区域和可能出现逆转的位置：
  - 密集成交区
  - 供应区（实测经常有效）
  - **第一波冲高后留下的上影线中部附近**
- TradingView筛选器中K线形态选**长上影线**（15分钟或1小时周期）
- 不用设置指标 — 山寨币变化太快，指标常滞后

**第四步：观察交易量动态入场** (02:31)
- 密切关注价格暴涨到关键阻力区时的交易量动态
- 高交易量 = 这个价格位置有活跃的多空交投行为 → 很可能大资金正在出货
- **交易量越高越好**
- 等多头攻势耗尽交易量后才入场
- **避免在第一根爆拉的K线上做空** — 多头攻势太强
- 不能看到风吹草动就开仓平仓
- 创新高币种上方没有参照区 → 危险，要等第一波结束+抛物线波动后再次到达关键区域时入场

## Exit & Stop Loss / 出场止损
**止损规则** (07:28)
- 最终止损设置必须是**强制性的前一个波段高点上方止损**
- 止损不能模糊，一定要硬性清楚
- 止损幅度可以放宽（容错率增加），但相应**减小开仓仓位**
- **止损幅度上限：不能超过两天的平均盈利**
  - 例：近月日均赚1000，单次最大损失不超过2000

## Risk Management / 仓位
- 只用自己不感到害怕的资金量进行交易（情绪不易受波动影响）
- 每次进场前评估潜在风险与回报，只在回报明显高于风险时考虑
- 止损宽 → 仓位小；止损紧 → 仓位可适当大

## Examples / 实例
- 2022年单日最高盈利600多万美元
- TradingView筛选器可在近千个活跃币种中实时检索

## Caveats / 注意
- 山寨币波动幅度很大，经常被打损
- 三座坟墓：**冲动交易、过度交易、报复性交易**
- 市场上能盈利的只有不到10%的少数人，与大多数人相反才不容易犯错
- 创新高没有参照区时危险
- 不要在第一根爆拉K线上做空
- 该策略可以**留到熊市用，效果非常好**
