# 永续合约 + 币币现货 吃费率套利 — 半神半木夏

## TL;DR
持有现货 + 永续合约等价值反向开空，吃永续合约支付的资金费率。当资金费率为正时，做多的人要向空头支付费用，因此持有现货 + 永续空单可以收取一年大约 100% 量级的费率收益（在 0.1% 的稳定费率下）。是半神半木夏标志性的"吃费率"做法，类似存款吃利息但年化高出数十倍。

## Setup / 形态
- 永续合约价格与现货价格之间存在价差，资金费率是平衡价格的工具。
- 资金费率为正 → 多头向空头支付费用
- 资金费率为负 → 空头向多头支付费用
- 支付额 = 仓位 × 资金费率
- OKX 永续合约每 8 小时结算一次（UTC 0/8/16 点）

## Entry Rules / 入场
1. 选择费率长期稳定（不频繁正负翻转）的大市值币种，如 BTC、ETH 及主流币（03:34）。可在 OKX 数据页面查历史资金费率。
2. 不选小市值币种，因为：
   - 费率正负翻转频繁，疏忽就反被吃费率（03:46）
   - 深度差，大仓位开/平容易出现成交风险（03:51）
3. 在永续合约费率为正时：
   - 持有等价值的币币现货（如 100 个 BTC）
   - 在永续合约开等价值的空单
4. 视频示例（00:51–01:09）：100 枚 BTC + 永续 100 BTC 空单，年化按 0.1% × 3 × 365 ≈ 109.5%（理论值，未扣手续费）。
5. 小资金（1000U）也可做，建议使用 OKX 的"双腿下单"策略减少操作成本与风险。

## OKX 双腿下单操作（02:01–03:11）
- 路径：交易 → 策略交易 → 策略广场 → 创建策略 → 组合套利 → 套利下单
- 选择套利标的（按资金费率从高到低排）
- 选择 USDT 保证金模式
- 因为是资金费率为正，所以左腿永续合约做空
- 价格选择市价（限价容易出现左右腿不同时成交的风险）
- 两腿数量输入相同（如均为 1000）
- 点击双腿下单，策略自动运行
- 必须在资金费用结算时点持仓才能收取费用，结算前平仓则本期不收取/支付任何费用

## Exit & Stop Loss / 出场止损
- 视频未给具体退出规则，本质为对冲套利（永续空 + 现货多，盈亏相抵），只要费率维持正向，可长期持有。
- 退出时机：当费率开始接近 0 或转负时（视频未明确，但暗示需"长期持有要选稳定费率币种"）。
- 风险来源不是市场价格，而是手续费 + 费率方向反转。

## Risk Management / 仓位
- 总仓位 = 现货价值 = 永续空单价值（盈亏相抵）
- 套利总收益公式（视频明确给出）：
  - 第一种（现货 + 永续）：`资金费收益 − 交易手续费 = 总收益`
  - 第二种（杠杆 + 永续）：`资金费收益 − 杠杆利息 − 交易手续费 = 总收益`
  - 第三种（交割合约 + 永续）：与第一种类似，省去杠杆利息
- 闲置资金都可做，没有强制最低门槛。

## Examples / 实例
- 100 枚 BTC 在 0.1% 稳定正费率下做一年：账户从 100 → ~109.5 枚 BTC（扣手续费后）
- 按 BTC 28,000 美金计算"这么大笔资金一年翻一倍"（00:55–01:15）

## Caveats / 注意
- 小币种费率高但风险高：一天 3 次结算，费率正负翻转你可能反过来被吃（03:46）
- 小币种深度差，大仓位进出有成交风险
- 必须在结算时点持仓才有费率收益
- 手续费 + （如使用杠杆的）杠杆利息会侵蚀收益，必须保证费率收益 > 这两项总和
- 视频强调这是与"存银行吃利息一样安全"的套利方式，但前提是配置合理
