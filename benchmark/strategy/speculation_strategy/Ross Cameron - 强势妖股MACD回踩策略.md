# 强势妖股MACD回踩日内策略 — Ross Cameron

## TL;DR
Ross Cameron用极简的MACD指标在当天最强、相对成交量最大、流通盘小的妖股上做日内突破回踩。核心：MACD本身不强，但放在强势股+大成交量+强情绪+小流通盘的环境下就非常好用。只做当天最强、最活跃、最有辨识度的股票。

## Setup / 形态
- 资产：美股（特别是低价小流通盘妖股）
- 时间框架：日内、分钟级别
- 工具：MACD指标 + Ross的选股滤网

## Entry Rules / 入场
**第一步：选股滤网** (01:09)
- 涨幅排行榜
- 相对成交量
- 流通盘
- 新闻催化
- 必须全部叠加才有意义

**关键过滤条件** (01:23)
- **相对成交量**：今天的成交量 ÷ 过去一段时间的平均成交量 ≥ **5倍**（越夸张信号越有用）
- **小流通盘 + 大成交量** = 大波动妖股最常见的底层结构
  - 例：Octo 流通盘只有268万股，当天成交量1.69亿股
- **新闻催化**

**第二步：确认MACD"打开"** (02:07)
- "打开"定义：MACD整体在0轴上方运行（柱状图 + 快慢线都在0轴上方）
- **最好是刚从0轴下方翻到上方** → 短线动能开始明显转强
- 这是值得交易的品种

**第三步：等价格回踩** (02:25)
- Ross的三种经常用的回踩形态：
  1. **Micro Pullback**：很小的回踩，可能就一两根K线，波幅很浅但节奏干净
  2. **牛旗形态**：前面一段强力拉升，后面走小幅震荡区间，然后再向上
  3. **ABCD结构**：先拉一段→回踩→再拉一段→整理→寻找下一次入场
- 这三种形态既是上涨中继结构，也帮我们在快速上涨中找到清晰的风险位置

## Exit & Stop Loss / 出场止损
**进场前先算盈亏比** (02:59)
- 例：Octo进场在大约8.9附近，止损8.66附近，每股风险0.25，做10000股先算清楚
- **如果目标位置连2:1都达不到，这笔交易质量就一般，不做**
- 算盈亏比的习惯让你进场前清楚是否值得冒险

**止损位置** (03:23)
- 回踩结构的低点下方
- 或那根突破K线的低点下方
- 跌破=回踩或突破大概率不是整理，而是转弱或清洗流动性，单子继续拿没有意义

**止盈：跟着走势灵活** (03:38)
- 不要死盯固定目标
- 参考：前高、整数关口、关键阻力位
- 真正决定出场的是盘中K线质量和动能状态
- 价格继续向上+成交量继续放大+K线强势 → 继续拿
- 冲高后出现很长上引线 / 几根质量差见顶味重的K线 → 先止盈

## Risk Management / 仓位
- 每股风险计算 + 仓位规模 → 总风险敞口
- 例：每股0.25风险 × 10000股 = 2500美金风险

## Examples / 实例
- **Octo**：当天最高涨幅超过5700%，Ross几小时赚31000美金，流通盘268万股，成交量1.69亿股
- **LCFY**：当天涨294%，成交量约2100万股，流通盘只有100万 → Ross当天总利润约77000美元，其中51000来自LCFY

## Caveats / 注意
- 这策略只适合当天最强、最活跃、最有辨识度的股票
- 普通股、横盘股、成交量很低的股都不在这个交易范围
- MACD本身没那么强，强势股+强势信号败乐才会加倍
- 短线交易最怕"买错了不肯走"，把小亏扛成大亏
