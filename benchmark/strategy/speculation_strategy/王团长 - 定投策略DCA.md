# 定投策略 DCA — 王团长

## TL;DR
固定时间间隔用固定金额买入加密资产，不论行情涨跌到点就买。核心信条："每周五定时定量投资，雷打不动"。利用均值回归摊薄成本，让无法择时和克服情绪的人也能盈利。

## Setup / 形态
- 标的：加密货币（推荐大市值币种）
- 周期：position（长期持有，按日/周/月定投）
- 适用：抓不准入场时机、不能坚定止盈止损、无法克服贪婪恐惧的人
- 王团长成名战绩：2017 年 100 万做到 900 万

## Entry Rules / 入场

### 策略本质
- 全称：Dollar Cost Average（DCA），美元平均成本
- 固定时间间隔内用固定金额买入某笔或几笔标的
- 不论行情涨跌，到点就买

### 与马丁格尔区别（00:46-01:07）
- 马丁格尔：固定仓位拆分，价格每下跌一个幅度比例就成倍数加仓，反弹达到止盈比例后全部卖出
- 定投：完全机械化，到点就买

### 数学示例（01:15-01:40）
- 设定每周用 1000 美元买 BTC
- BTC = 20000U 时 → 买 0.05 个
- BTC 跌到 15000U → 1000 美元可以买 0.067 个
- 经过时间推移利用市场上下波动有效拉低持仓成本

### OKX 操作步骤（01:40-03:36）
1. **入口**：OKX 策略广场 → "平均成本" 菜单栏 → "定投策略" → 创建策略
2. **参数 1：币种配置**
   - 可选某一种加密货币定投
   - 可选几个币种组合一起定投
3. **参数 2：时间和金额**
   - 定投周期：按日 / 周 / 月
   - 投入时刻：定投当日买入的具体时刻（如中午 12:00）
   - 投入金额：根据经济状况和风险承受能力（示例 1000U）

### 币种选择规则（02:17-02:33）
- **选大市值币种**：比特币、以太坊、莱特币
- 既有预期也有深度，不会跌的太离谱
- **避免小市值山寨币**：买到就是被套到，定投就成了定期被套

## Exit & Stop Loss / 出场止损
**关键卖出原则**（03:51-03:59）：
- 设置一个不能太难达到的目标
- 一定要在达到"买了也觉得不会亏的位置"出手
- 不能留到鱼尾行情
- 否则容易被带入下一个牛熊周期
- 没有获利了结的盈利总归是浮盈

## Risk Management / 仓位
- 投入金额要让自己在长期内持续投资
- 不会给日常生活造成过大负担
- 利用均值回归让风险分散

## Caveats / 注意

**微笑定投曲线 vs 哭泣曲线**（04:22-04:53）：

**微笑曲线（成功前提）**：
- 选择的标的处于长期向好的大趋势
- 买入后整体趋势走低 → 以更低成本积累更多资产
- 市场再次上涨时资产迅速增加

**哭泣曲线（失败模式）**：
- 资产长期表现不乐观
- 开始价格上升后购买成本不断叠加
- 一旦之后市场大跌 → **亏损甚至比一次性买入还要多**

**核心要点**：
- 选股一定要选长期向好的大趋势
- 必须设定止盈目标，到点就卖
- 历史回测不代表未来，但过去赚到过钱的币总比从没赚到过钱的币靠谱一些
