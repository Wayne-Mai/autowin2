# 抛物线做空模型 — Marios Stamatoudis

## TL;DR
Marios三种模型之三。标的价格在短时间内出现剧烈抛物线式上升（单日30%/40%/50%或几天翻倍）后，出现红色下跌日或巨大上影线遇阻回落 → 第二根K线到达上影线一半位置时入场做空。

## Setup / 形态
- 资产：股票（市值低于10亿美元的小市值股）
- 时间框架：日线图
- 工具：观察单日大幅涨幅和巨大成交量

## Entry Rules / 入场
**模型条件** (03:09)
- 日线图标的价格在**短时间内出现剧烈抛物线式上升**
- 例如：单日涨了30%/40%/50%的涨幅
- 或几天内翻了一倍还多
- 连续多天上涨后**出现巨大成交量** → 上涨期间消耗了大量买盘

**入场触发** (03:41)
- 趋势减弱出现**红色下跌日**
- 或出现**巨大的上影线遇阻回落**
- 第二根K线**再到达上影线一半的位置**时 → 入场空单

## Exit & Stop Loss / 出场止损
- 抛物线模型交易风险较大
- **不要留单子过夜**
- 除非吃到了瞬时利润 → 将剩余部分仓位设置成本价止损
- 这样风险变得可控

## Risk Management / 仓位
- 同时持有6-7个品种（不超过15个）
- 平均仓位13-16%
- 每笔风险0.25-0.4%
- 这种逆势空单风险大，仓位要严格控制

## Examples / 实例
- **PKST**：市值低于10亿美元的标的，五天的时间价格上涨400%以上 → Marios抛物线做空 → 获利超过35%

## Caveats / 注意
- 风险较大，绝对不要留单子过夜
- 必须等第二根K线到达上影线一半位置才入场
- 不能在第一根抛物线K线上做空
- 小市值标的容易被操纵，风险高
