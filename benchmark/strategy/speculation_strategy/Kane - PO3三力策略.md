# PO3三力策略（积累-操纵-派发） — Kane

## TL;DR
Kane的PO3 (Power of Three) 模型把价格运动拆为积累(横盘)-操纵(扫流动性)-派发(快速反转回中轴)三阶段。通过日线/4小时/1小时三周期共振 + SMT (Smart Money Technique Divergence) 跨品种背离 + 反转区拒绝，捕捉操纵高/低点之后迅速反转回区间50%中轴的那段"最干净的基础获利"。属于反转策略，目标不是吃整段趋势，胜率高、信号清晰、风险明确。

## Setup / 形态
- PO3三阶段 (00:55):
  - 积累 (Accumulation)：价格横盘
  - 操纵 (Manipulation)：价格向上或向下扫掉横盘区间外的流动性
  - 派发 (Distribution)：价格反转并快速回到50%中轴线
- 基础市场特性 (01:06)：在延续主要趋势之前，价格通常会先回撤到前一区间的50%中轴线再重新启动。
- 模型重点捕捉：价格在操纵某个关键高点或低点后、迅速反转并重返区间的那一刻 (01:21)。
- 适用：纳指期货 (Kane主做)、标普500、加密货币 (BTC等)。

## Entry Rules / 入场

### 第一步 — 多周期方向共振 (01:47)
1. **日线图**：标记前一交易日高点、低点，并划出大区间的50%中点；提供方向性指导和目标区。
2. **4小时图** (02:00)：要看到价格先冲上一个高溢价区扫过前高（操纵阶段完成），随后明显回落（派发进行中）；结构要清晰，体现出PO3三阶段：
   - 吸筹阶段：价格在之前构建的横盘区间
   - 操纵阶段：价格突破前高、触发流动性
   - 分配阶段：从高溢价区域被快速打回回落
3. **1小时图** (02:34)：观察是否形成同样PO3结构。
4. 日线、4小时、1小时三周期都对齐，才可以在这个基础上寻找入场点 (02:39)。

### 第二步 — 等待 SMT 背离 (02:45)
- SMT = Smart Money Technique Divergence (02:50)：不同但高度相关的市场或品种在同一时间段内没有走出相同的走势。
- Kane配对 (03:03)：纳指期货 vs 标普500期货 同时观察；加密货币市场可用 BTC 与高相关性币种做比对 (04:05)。
- 逻辑 (03:11)：大资金在两个相关市场同时推高的难度远大于单独市场；如果一个品种创新高、另一个没跟上，那个创新高大概率是假突破/操纵行为。
- **关键时间点** (03:27)：留意美国东部时间上午10点（北京时间晚10-11点）= 美股开盘半小时后，机构常完成第一轮建仓/调整，是短线流动性极值点，扫单/操纵动作概率尤其高。
- **做空场景** (03:51)：价格震荡后先冲破前高、然后快速回落，确认SMT背离（纳指新高而标普没同步），即为反转信号。

### 第三步 — 寻找反转区 (04:10)
- 价格常从之前的不平衡区反应：旧支撑转新阻力 (04:15)，这种区域非常容易出现拒绝行为，是入场重点观察区。
- 三共振条件全部具备：多图表PO3结构 + SMT背离 + 价格处于反转区。

### 第四步 — 切小周期确认入场 (04:33)
- 切到 5分钟图 或 15分钟图 找具体入场点。
- 两种入场方式：
  - 跌破并收盘在反转区下方时入场 (04:42)
  - 回踩反转区失败时挂单入场 (04:46)

## Exit & Stop Loss / 出场止损
- **止损** (04:49)：设置在 SMT 高点（做空）或 SMT 低点（做多）的附近。
- **止盈** (04:54)：1小时交易图表的50%中轴线；目标是快速拿到基础获利，而不是贪图整段趋势 (04:58)。
- **保本移动** (05:02)：当价格朝有利方向运行并确认后，立即将止损移动到保本规避风险。
- **策略失效退出** (05:08)：如果价格重新回到操纵高点或低点以上，策略失效要及时止损出场。

## Risk Management / 仓位
- 入场胜率高，风险明确 (01:27)；属于反转策略，意味着是在博弈趋势的短期拐点 (05:16)，等于和当前动量对着干，没有明确确认信号贸然进场风险很大。
- 必须具备耐心，等待信号完全成型再行动 (05:24)。
- 模型启动后通常能在日内完成交易，效率极高 (05:32)。

## Examples / 实例
- 案例 (03:51) 做空：价格震荡后冲破前高、快速回落，纳指创新高而标普没有同步（SMT背离），切到5分钟图找入场点。

## Caveats / 注意
- 反转策略，是和当前动量对着干 (05:19)，必须等信号完全成型。
- 关键时间点（美东上午10点）是高频出现操纵动作的窗口 (03:30)，其他时间频率较低。
- 跨品种背离在加密市场需要找高相关性币对 (04:06)。
- 策略目标只吃从操纵区返回中轴的这一段，不要贪整段趋势 (01:29)。
