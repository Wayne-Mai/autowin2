# 均值回归震荡模型 — Fabio Valentini

## TL;DR
Fabio 的两个高频模型之一，专用于**震荡盘整市场**。等待突破盘整失败 → 标记突破波段的低成交量节点 → 观察主动回归订单入场 → 止盈到当前盘整区的 POC。适合伦敦时段或波动小时段。

## Setup / 形态
- **适用时段**：**伦敦时段**这种波动受限的时段，或波动较小的市场（00:03:43-03:45）
- **理由**：市场常态更多是震荡盘整，趋势模型在震荡中容易频繁止损
- **核心工具**：
  - VP 成交量分布图
  - 订单流 / 订单簿
  - POC（控制点）

## Entry Rules / 入场

**步骤1 — 识别市场是否正在盘整**（00:03:45-03:57）：
- 标记出**清晰的盘整区间**
- 用成交量分布图覆盖这个区间
- 界定出**明确的盘整区**和**控制点 POC**

**步骤2 — 识别入场结构（突破失败）**（00:03:57-04:16）：
- **不能**在价格第一次触及或突破盘整边界时入场
- 等待一个特定的**突破失败结构**：
  - 价格先**突破盘整区失败**
  - 突破未能延续 → 表示动能不强
- 标记**这次突破的发起点 → 突破顶点**
- 用 VP 标出这个波段的成交量分布图

**步骤3 — 入场**（00:04:16-04:23）：
- 当价格回到**低成交量节点 LVN** 的位置
- 观察是否有**与回归方向一致的主动订单**
- 这是大资金在推动价格回归的痕迹

## Exit & Stop Loss / 出场止损
**止损**（00:04:23-04:31）：
- 设置在**主动大单的后方**
- 加上一些小的缓冲空间

**止盈**（00:04:35-04:44）：
- 设置在**当前盘整区间的控制点 POC 位置**
- 因为 POC 是交易量最密集的位置 → 均值回归到这里的概率最高

## Risk Management / 仓位
- 每笔交易风险同样**不超过账户资金的 0.5%**（00:04:32）
- A/B/C 三级风险管理同上模型（见配套文件）
- A级 €2500-3000；B级 ≤€2000；C级 €1000-1500
- 一天开始时低仓位，随当日盈利积累再增加风险资金

## Caveats / 注意
- **必须等突破失败结构**，不能在边界第一次触及就反向交易
- 不适用于明确趋势行情
- 通过趋势跟踪 + 均值回归模型结合运用，"不再容易被市场情绪左右，而是主动跟随市场真实节奏"（00:05:25-05:34）
- 每个交易员应找到自己擅长的细分领域，深入了解资产特性、波动性、触发价格变动的潜在规律
- Fabio 自述同时交易多种资产没赚到钱，专注于一两种后才显著提升
