# PA+MACD 策略 — Darek Dargo

## TL;DR
波兰交易员 Darek 用 4 小时供需区/支撑阻力 + 孕线/吞没/Pinbar 价格形态 + MACD 能量柱颜色翻转 三步验证入场。1000 笔实盘 84% 胜率，2023 上半年达 95%。

## Setup / 形态
- 标的：外汇、股票、加密货币（mixed）
- 主周期：4 小时或以上
- 经历：2008 年关闭进出口公司全职做交易；曾用十几种指标决策很乱；发现 PA+MACD 后命运齿轮转动
- 核心理念：少数确定性高的指标共振，远胜十几个指标决策

## Entry Rules / 入场

### 第一步：标记关键区域（4h+ 周期）（02:39-04:10）

**供需区概念**：
- 大资金机构推动价格大幅波动
- 通常在某些区间布置大量订单
- 订单只成交一部分行情就启动 → 大资金会等价格再次回到目标区间继续交易
- 很少像散户追涨杀跌

**需求区两种结构**：
1. **跌平涨**：市场先下跌 → 走平 → 上涨
   - 行情反转前的小平台节点 = 需求区
   - 价格再次靠近时未成交订单可能再次阻止下跌
2. **涨平涨**：市场先上涨 → 走平 → 继续上涨
   - 表示一段行情中短暂的多空换手
   - 上涨中的平台节点 = 另一种需求区

**供给区两种结构**：
1. **涨平跌**
2. **跌平跌**
- 标记方式和判定方法与需求区相同

### 第二步：价格形态验证（04:10-04:23）
- 价格在潜在翻转区域附近**必须出现**：
  - 标准的**孕线**
  - 或**吞没**形态
  - 或 **Pinbar**（多头规则中提到）

### 第三步：MACD 能量柱验证（04:23-04:57）

**指标设置**：
- 搜索栏输入 MACD 或 平滑异同平均线
- 添加到图表
- **MACD 和信号线都勾掉**
- 只留下**能量柱**（双色）

**作为振荡器使用**：
- 只用双色能量柱印证方向变化
- **绿转红柱 = 多转空**
- **红转绿柱 = 空转多**
- **波峰波谷必须远离零线，不能太平坦**
- **波动要规律**，不能一直红绿红绿不规则变动

### 完整入场规则

**做空规则**（05:00-05:30，BTC 4h 示例）：
1. 一段上涨走势中
2. 价格到达**供给区**附近
3. 出现**孕线**形态（说明市场犹豫，价格小范围盘整酝酿下一步动作）
4. 价格向下突破孕线**低点**收盘
5. 此时 MACD **从绿色柱转成红色柱**
6. 条件全部满足 → 在收盘价入场空单

**做多规则**（05:30-05:58）：
1. 一段下跌走势中
2. 价格到达**需求区**附近
3. 出现标准的 **Pinbar 或吞没**形态（说明未成交订单或主动买单太多，价格难一次跌破）
4. 价格留下**长下影线**并向上突破前一根 K 线高点
5. 此时 MACD **从红色柱转成绿色柱**
6. 条件全部满足 → 在收盘价入场多单

## Exit & Stop Loss / 出场止损

**止损位置**（06:01-06:10）：
- 空单：供给区**上方一些**
- 多单：需求区**下方附近**
- 如果区域范围太大 → 放到入场 K 线的末端

**止盈风格**（06:10-06:28）：
- **不会刻意追求 1:2 或更高的风险回报比**
- 因为 4 小时图表止损幅度通常都不小
- 空间太大 → 风险和不确定性因素也变多
- 更倾向于在**临近的关键区域附近止盈**
- 风险回报比经常在 **1:1 以下**
- 自认胜率足够高也足够稳定 → 不需要高盈亏比

## Caveats / 注意
- 三个条件必须**全部满足**才入场（高胜率代价是低频）
- 4 小时图止损幅度本身就不小，所以不强求高盈亏比

- 不规则的红绿变动不要交易
- 跨市场通用（外汇 / 股票 / 加密）
