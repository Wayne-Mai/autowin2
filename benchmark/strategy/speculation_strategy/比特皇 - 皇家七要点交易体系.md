# 比特皇皇家七要点交易体系 — 比特皇

## TL;DR
比特皇用1500美元做到2397万美元的合约复利交易体系，由七大要点组成：确定交易级别、试仓三种高胜率形态、加仓规则、止损止盈、仓位与杠杆、心态控制、复盘。核心是顺势交易、严格止损（盈亏比至少10:1）、加仓只在浮盈时进行、随资金体量上升不断降低杠杆。

## Setup / 形态
- 资产：加密货币合约（比特币为主）
- 时间框架：根据交易级别灵活选择，2小时/日线/月线
- 重要思想：所有交易都是趋势交易，只是趋势级别不一样 (01:37)。小级别就要做短线、长拿就容易利润回撤；大级别就要适当把止盈止损都放宽，做好长时间持仓的准备 (01:46)
- 警示：短线长拿、长线短拿这种没原则的持仓非常危险 (01:57)

## Entry Rules / 入场
**要点一：确定交易级别** (01:20)
- 先判断当前操作的是哪个级别趋势，再决定开单和止盈止损位置的设置

**要点二：试仓与三种高胜率形态** (02:00)
- 出现可能30%以上波动机会时，耐心等可能出现转折的地方，只要有就先开进去试仓 (02:06)
- 级别越大，胜率就越高
- **形态1：看涨/看跌三角形态** (02:21) — K线盘整过程，看涨三角是空军节节败退，看跌三角是空军占优
- **形态2：盘整筑底形态** (02:38) — 大级别下跌中的盘整，B点过A点，第二次下跌D点没过C点，更高高点+更高低点意味着底部大概率反转；反之B点低于A点则大概率是下跌中继
- **形态3：下跌斜率+交易量结合的筑底形态** (03:04) — 交易量缩小=多头/空头衰竭；下跌斜率放缓+成交量逐渐减少=卖出资金减少，这也是筑底

**要点三：开单加仓** (03:26)
- 开单分两种：不确定时小区间分批开单；确定性高时一次性开满
- 加仓只在大级别趋势中考虑
- 关键规则：**加仓只在浮盈的时候才加** (04:00)
- 加仓部分一般不会超过底仓的1倍
- 加仓的部分也要设置止损
- **亏损时不能加仓**，不然总有一次会让你损失惨重 (04:13)

## Exit & Stop Loss / 出场止损
**要点四：止损止盈** (04:13)
- 止损放在第一位，开单前首先考虑离止损位有多远，止损幅度是否在承受范围内 (05:21)
- 连续止损处理法 (04:42)：
  - 第一次止损3成仓位被止损 → 第二次用3.2成 → 第三次用3.5成
  - 第三次仍止损则**无条件强制休息**
  - 好处：做对一次可以把前面的损失补回来
- **三种止盈方式**：一次性止盈、分批止盈、对冲 (05:30)
  - 走势不符合预期或达到止盈位置 → 一次性止盈
  - 行情发展中遇到异常走势 → 分批止盈或对冲
  - 止盈位置可根据压力支撑、各种指标判断
- **盈亏比至少10:1以上** (06:02)
- 止损痛一次要比浮亏持续的痛苦好得多 (05:18)

## Risk Management / 仓位
**要点五：仓位和杠杆** (06:06)
- 不同行情仓位大小要有区别，遇到大行情敢于重仓操作
- 比特皇资金成长各阶段的杠杆 (06:25)：
  - 1万本金 → 第一笔20倍杠杆做多BTC，半个月滚到十几万
  - 资金几十万：不超过10倍杠杆
  - 资金两三百万：基本不超过5倍杠杆
  - 大资金量后：平时只开3倍左右，且只占总仓位的一到三成
- 杠杆选择取决于：风险偏好、合约币种、合约资金多少、单利还是复利、行情大小 (06:50)
- 小资金图高收益：根据止损位计算，杠杆越高越好，不这样很难赚到第一桶金 (07:07)
- 单利与复利区别 (07:12)：单利可一直保持一定杠杆倍数；做复利时随资金体量上升杠杆一定要降低，提高容错率

## Examples / 实例
- 比特皇从1500美元到2397万美元（约1亿7000多万人民币）
- 第一笔交易20倍杠杆做多比特币，半个月滚仓到十几万

## Caveats / 注意
**要点六：心态** (07:26)
- 大幅回撤的主要原因是"心浮气躁"
  - 心浮 = 大幅盈利后盲目自信，骄兵必败 (07:33)
  - 气躁 = 连续止损后心态崩溃，开单已不理智 (07:37)
- 交易纪律：**当天连续止损三次就无条件停止交易一天** (07:46)
- 把自己锻炼成机器，不带情绪交易，但人性的贪嗔痴一生都需要修行 (07:56)

**要点七：复盘** (08:06)
- Livermore靠历史行情记忆预期未来走势
- 比特皇能猜出19年至今的任意30分钟以上K线段是哪段时间
- "书读百遍其义自现"，复盘时间久了就会锻炼出盘感
- 二级市场反应的是人性的贪婪与恐惧
- 交易没有捷径，需要大量练习
